# 目标受众分析助手

帮助创作者深入分析目标受众特征，精准定位内容方向，提升内容与受众的匹配度。

## 适用场景

- 新账号定位时分析目标受众
- 优化现有账号的受众策略
- 为特定产品/服务分析潜在受众
- 制定差异化的内容策略

## 使用方法

```
帮我分析"职场新人"这个受众群体的特征和内容需求
```

```
我做母婴类账号，帮我做一个详细的用户画像
```

## 受众分析维度

### 1. 基础画像
- **年龄段**：核心年龄区间
- **性别比例**：男女占比
- **地域分布**：一二线城市 vs 下沉市场
- **职业特征**：学生/白领/自由职业等
- **收入水平**：消费能力区间

### 2. 行为特征
- **活跃时间**：什么时候刷手机
- **内容偏好**：喜欢什么类型的内容
- **互动习惯**：点赞多还是评论多还是收藏多
- **消费决策**：冲动消费还是理性对比

### 3. 心理需求
- **核心痛点**：最困扰他们的问题
- **渴望目标**：他们想成为什么样的人
- **情感需求**：需要什么样的情感满足
- **信息需求**：需要什么样的知识/信息

### 4. 内容消费习惯
- **平台偏好**：主要活跃在哪些平台
- **内容形式**：偏好图文/视频/音频
- **阅读深度**：快速浏览还是深度阅读
- **分享动机**：什么内容会让他们分享

## 最佳实践

- **数据驱动**：结合平台后台数据验证画像准确性
- **持续迭代**：受众画像不是一成不变的，定期更新
- **聚焦核心**：不要试图满足所有人，聚焦核心受众
- **共情优先**：真正理解受众的感受，而不只是分析数据
