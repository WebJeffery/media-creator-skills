# 受众分析参考资料

## 受众分析方法

### 数据分析法

| 数据来源 | 可获取信息 | 分析方法 |
|----------|------------|----------|
| 平台后台 | 粉丝画像、活跃时间 | 直接查看 |
| 评论区 | 用户需求、痛点 | 内容分析 |
| 私信 | 具体问题、反馈 | 归类整理 |
| 竞品 | 目标用户特征 | 对比分析 |

### 调研访谈法

| 方法 | 优点 | 缺点 |
|------|------|------|
| 问卷调研 | 样本量大、可量化 | 深度不足 |
| 深度访谈 | 深入了解、发现洞察 | 样本量小 |
| 焦点小组 | 互动讨论、激发想法 | 组织成本高 |
| 用户观察 | 真实行为、无偏差 | 耗时长 |

## 用户画像维度

### 人口统计

| 维度 | 细分 |
|------|------|
| 年龄 | 18-24/25-30/31-40/40+ |
| 性别 | 男/女 |
| 地域 | 一线/新一线/二线/三线及以下 |
| 职业 | 学生/白领/自由职业/企业主 |
| 收入 | 低/中/高 |
| 教育 | 高中/大专/本科/研究生 |

### 心理特征

| 维度 | 说明 |
|------|------|
| 价值观 | 用户看重什么 |
| 生活方式 | 用户如何生活 |
| 兴趣爱好 | 用户喜欢什么 |
| 消费观念 | 用户如何消费 |

### 行为特征

| 维度 | 说明 |
|------|------|
| 媒体习惯 | 使用什么平台、什么时间 |
| 内容偏好 | 喜欢什么类型的内容 |
| 互动习惯 | 点赞/评论/分享的习惯 |
| 消费行为 | 购买决策过程 |

## 各平台用户特征

### 公众号用户

| 特征 | 描述 |
|------|------|
| 年龄 | 25-45岁为主 |
| 行为 | 深度阅读、订阅习惯 |
| 偏好 | 干货、深度内容 |
| 活跃时间 | 早晚通勤、午休 |

### 小红书用户

| 特征 | 描述 |
|------|------|
| 年龄 | 18-35岁为主 |
| 性别 | 女性占比高（70%+） |
| 行为 | 种草、决策参考 |
| 偏好 | 真实分享、颜值内容 |

### 抖音用户

| 特征 | 描述 |
|------|------|
| 年龄 | 全年龄段覆盖 |
| 行为 | 碎片化娱乐、冲动消费 |
| 偏好 | 有趣、有用、有共鸣 |
| 活跃时间 | 午休、晚间 |
