# 受众分析模板

## 用户画像模板

```markdown
# 目标受众画像

## 基础信息

| 维度 | 描述 |
|------|------|
| 年龄 | {年龄范围} |
| 性别 | {男/女/不限} |
| 地域 | {城市级别/地区} |
| 职业 | {职业类型} |
| 收入 | {收入水平} |
| 教育 | {学历水平} |

## 行为特征

### 内容消费习惯
- 活跃时间：{时间段}
- 常用平台：{平台列表}
- 内容偏好：{内容类型}
- 消费时长：{时长}

### 消费决策特征
- 决策因素：{因素}
- 价格敏感度：{高/中/低}
- 品牌忠诚度：{高/中/低}
- 决策周期：{时长}

## 心理特征

### 痛点
1. {痛点1}
2. {痛点2}
3. {痛点3}

### 需求
1. {需求1}
2. {需求2}
3. {需求3}

### 恐惧
1. {恐惧1}
2. {恐惧2}

### 渴望
1. {渴望1}
2. {渴望2}

## 内容策略建议

| 维度 | 建议 |
|------|------|
| 内容类型 | {建议} |
| 表达风格 | {建议} |
| 发布时间 | {建议} |
| 互动方式 | {建议} |
```

## 受众调研模板

```markdown
# 受众调研问卷

## 基础信息
1. 您的年龄是？
2. 您的职业是？
3. 您所在的城市？

## 内容偏好
4. 您通常在什么时间刷{平台}？
5. 您最喜欢看什么类型的内容？
6. 您关注账号的主要原因是？

## 痛点需求
7. 在{领域}方面，您最大的困扰是？
8. 您希望从内容中获得什么？
9. 什么样的内容会让您点赞/收藏？

## 开放问题
10. 您还有什么想说的？
```

## 竞品受众分析

```markdown
# 竞品受众分析

## 竞品账号
- 账号名：{名称}
- 粉丝数：{数量}
- 定位：{定位}

## 受众特征分析

### 从评论区分析
- 用户称呼：{如何称呼博主}
- 常见问题：{用户问什么}
- 情感倾向：{正面/负面}

### 从内容分析
- 高互动内容：{类型}
- 低互动内容：{类型}
- 用户偏好：{偏好}

## 差异化机会
- {机会1}
- {机会2}
```
