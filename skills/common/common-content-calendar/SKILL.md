# 多平台内容日历规划

帮助新媒体创作者统一管理公众号、小红书、抖音等多平台的内容发布节奏，生成结构化的内容日历。

## 适用场景

- 制定多平台联动的内容发布计划
- 规划周/月/季度内容排期
- 协调不同平台的内容差异化策略
- 管理内容生产进度

## 使用方法

```
帮我做一个下周的内容日历，我运营公众号和小红书，方向是"个人成长"
```

```
制定一个月度内容计划，覆盖公众号、小红书和抖音，主题是"健康饮食"
```

## 各平台发布节奏建议

| 平台 | 建议频率 | 最佳发布时间 | 内容特点 |
|------|----------|-------------|----------|
| 公众号 | 2-4 篇/周 | 8:00、12:00、20:00 | 深度长文、干货分享 |
| 小红书 | 3-5 篇/周 | 12:00-14:00、18:00-22:00 | 图文种草、生活分享 |
| 抖音 | 5-7 条/周 | 12:00-13:00、18:00-20:00 | 短视频、知识口播 |

## 内容复用策略

一个选题可以在多平台差异化发布：

```
选题："5个高效学习方法"

公众号 → 3000字深度长文，含案例和数据
小红书 → 图文笔记，配精美信息图，口语化表达
抖音   → 60秒口播视频，每个方法10秒速讲
```

## 最佳实践

- **一鱼多吃**：一个选题多平台差异化发布，最大化内容价值
- **错峰发布**：不同平台错开发布时间，避免自己和自己抢流量
- **预留弹性**：留出 1-2 个空位用于追热点
- **复盘迭代**：每周复盘数据，调整下周计划
