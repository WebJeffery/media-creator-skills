# 经典文案公式集合

收录新媒体创作中最实用的文案公式，帮助创作者快速产出高转化率的文案。

## 适用场景

- 撰写产品推广文案
- 编写广告投放素材
- 优化落地页/详情页文案
- 提升内容的说服力和转化率

## 使用方法

```
用AIDA公式帮我写一段护肤品推广文案
```

```
用PAS公式写一段关于时间管理课程的卖点文案
```

## 核心文案公式

### 1. AIDA 公式（注意→兴趣→欲望→行动）

| 步骤 | 说明 | 示例 |
|------|------|------|
| Attention | 抓住注意力 | 「90%的人都不知道的护肤真相」 |
| Interest | 激发兴趣 | 「原来这个成分才是抗老的关键」 |
| Desire | 制造欲望 | 「用了2周，同事问我是不是打了水光针」 |
| Action | 引导行动 | 「点击链接，限时8折」 |

### 2. PAS 公式（问题→激化→解决）

| 步骤 | 说明 | 示例 |
|------|------|------|
| Problem | 提出问题 | 「每天加班到10点，完全没有自己的时间」 |
| Agitate | 激化痛点 | 「长期这样下去，健康、家庭都会出问题」 |
| Solution | 给出方案 | 「这套时间管理方法，帮你每天多出2小时」 |

### 3. FAB 公式（特征→优势→利益）

| 步骤 | 说明 | 示例 |
|------|------|------|
| Feature | 产品特征 | 「含有5%烟酰胺成分」 |
| Advantage | 功能优势 | 「能有效抑制黑色素生成」 |
| Benefit | 用户利益 | 「坚持使用28天，肤色提亮一个度」 |

### 4. SCQA 公式（情境→冲突→问题→答案）

| 步骤 | 说明 | 示例 |
|------|------|------|
| Situation | 描述情境 | 「你每天花2小时通勤」 |
| Complication | 引入冲突 | 「但这2小时你只是在刷手机」 |
| Question | 提出问题 | 「如何把通勤时间变成成长时间？」 |
| Answer | 给出答案 | 「试试这3个通勤学习法」 |

### 5. 4U 公式（紧迫→独特→具体→有用）

| 要素 | 说明 | 示例 |
|------|------|------|
| Urgent | 紧迫感 | 「限时3天」 |
| Unique | 独特性 | 「独家配方」 |
| Ultra-specific | 具体化 | 「提升37%的转化率」 |
| Useful | 实用性 | 「立即可用的模板」 |

### 6. BAB 公式（之前→之后→桥梁）

| 步骤 | 说明 | 示例 |
|------|------|------|
| Before | 现状痛点 | 「以前写文章要憋一整天」 |
| After | 理想状态 | 「现在1小时就能写完一篇爆款」 |
| Bridge | 解决桥梁 | 「只因为我学会了这套写作框架」 |

## 输出格式

每个文案包含：
1. 使用的公式名称
2. 完整文案内容
3. 各步骤拆解说明
4. 适用平台建议

## 最佳实践

- **公式是骨架**：公式提供结构，但内容要有血有肉
- **组合使用**：多个公式可以组合，如 AIDA + FAB
- **口语化改造**：公式产出的文案需要口语化润色
- **A/B 测试**：同一产品用不同公式写文案，测试效果
- **真实感**：加入真实体验和细节，避免模板感太强
