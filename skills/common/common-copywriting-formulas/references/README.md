# 文案公式参考资料

## 公式适用场景

| 公式 | 最佳场景 | 特点 |
|------|----------|------|
| AIDA | 销售文案、广告 | 完整转化路径 |
| PAS | 痛点营销、问题解决 | 情感驱动 |
| FAB | 产品介绍、功能说明 | 逻辑清晰 |
| SCQA | 故事叙述、观点输出 | 结构完整 |
| 4U | 标题、短文案 | 快速吸引 |
| BAB | 效果展示、案例分享 | 对比强烈 |

## 公式组合使用

### AIDA + FAB

```
适用：产品详情页、种草文案

结构：
1. Attention：痛点开头
2. Interest：FAB介绍产品
3. Desire：效果展示
4. Action：购买引导
```

### PAS + SCQA

```
适用：深度内容、观点文章

结构：
1. Problem/Situation：描述问题
2. Agitate/Complication：分析原因
3. Question：提出核心问题
4. Solution/Answer：给出方案
```

## 经典文案案例

### AIDA 案例

```
【Attention】
"90%的人都不知道，你的护肤方法可能是错的"

【Interest】
"皮肤科医生告诉我，很多人的护肤顺序完全反了..."

【Desire】
"用了正确方法后，我的皮肤在2周内明显改善..."

【Action】
"点击下方链接，获取完整护肤指南"
```

### PAS 案例

```
【Problem】
"每天加班到10点，完全没有自己的时间"

【Agitate】
"长期这样下去，健康会出问题，家庭会有矛盾，
你的人生就这样被工作吞噬了吗？"

【Solution】
"这套时间管理方法，帮你每天多出2小时，
既能完成工作，又能享受生活"
```

## 文案写作技巧

### 提升说服力

1. **使用具体数字**
   - ❌ "很多人"
   - ✅ "87%的用户"

2. **加入真实案例**
   - ❌ "效果很好"
   - ✅ "用户小王使用后..."

3. **制造紧迫感**
   - ❌ "欢迎购买"
   - ✅ "限时3天，错过等一年"

### 避免的问题

| 问题 | 示例 | 改进 |
|------|------|------|
| 太抽象 | "质量好" | "通过XX认证" |
| 太夸张 | "最好的" | "XX用户选择" |
| 太啰嗦 | 长段落 | 分点说明 |
| 太生硬 | 说明书式 | 口语化表达 |
