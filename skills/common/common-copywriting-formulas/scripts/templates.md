# 文案公式模板

## AIDA 公式

```markdown
# AIDA 文案模板

## Attention（注意）
{抓住注意力的开头}
- 震撼数据
- 痛点问题
- 悬念设置

## Interest（兴趣）
{激发兴趣的内容}
- 相关故事
- 深入分析
- 独特观点

## Desire（欲望）
{制造欲望的描述}
- 效果展示
- 用户证言
- 场景描绘

## Action（行动）
{引导行动的结尾}
- 明确指令
- 限时优惠
- 简化步骤
```

## PAS 公式

```markdown
# PAS 文案模板

## Problem（问题）
{描述用户面临的问题}
- 具体场景
- 真实痛点
- 引发共鸣

## Agitate（激化）
{放大问题的严重性}
- 后果描述
- 情感渲染
- 紧迫感

## Solution（解决）
{提供解决方案}
- 产品/方法介绍
- 效果说明
- 行动指引
```

## FAB 公式

```markdown
# FAB 文案模板

## Feature（特征）
{产品/服务的特征}
- 具体参数
- 技术特点
- 独特属性

## Advantage（优势）
{特征带来的优势}
- 功能优势
- 对比优势
- 技术优势

## Benefit（利益）
{用户获得的利益}
- 解决什么问题
- 带来什么好处
- 具体效果
```

## SCQA 公式

```markdown
# SCQA 文案模板

## Situation（情境）
{描述当前情境}

## Complication（冲突）
{引入冲突或问题}

## Question（问题）
{提出核心问题}

## Answer（答案）
{给出解决方案}
```

## 4U 公式

```markdown
# 4U 文案模板

## Urgent（紧迫）
{制造紧迫感}
- 限时
- 限量
- 即将结束

## Unique（独特）
{强调独特性}
- 独家
- 首创
- 唯一

## Ultra-specific（具体）
{具体化描述}
- 具体数字
- 具体效果
- 具体案例

## Useful（有用）
{强调实用性}
- 立即可用
- 解决问题
- 带来价值
```
