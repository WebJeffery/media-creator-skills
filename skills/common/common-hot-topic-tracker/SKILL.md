# 全网热点追踪助手

帮助创作者快速发现和分析全网热点，生成可落地的追热内容方案。

## 适用场景

- 发现当前全网热门话题
- 分析热点与自身账号的关联度
- 快速生成追热内容方案
- 评估热点的生命周期和追踪价值

## 使用方法

```
最近有什么热点可以追？我的账号方向是"职场成长"
```

```
分析一下"XXX事件"这个热点，我做美食账号能怎么追
```

## 热点评估框架

### 追热决策矩阵

| 维度 | 权重 | 评估标准 |
|------|------|----------|
| 热度 | 25% | 搜索量、讨论量、上升趋势 |
| 相关性 | 30% | 与账号定位的契合程度 |
| 时效性 | 20% | 热点剩余生命周期 |
| 安全性 | 25% | 是否涉及敏感话题、争议风险 |

### 热点分类

| 类型 | 生命周期 | 追踪策略 |
|------|----------|----------|
| 突发热点 | 1-3 天 | 快速响应，2小时内发布 |
| 话题热点 | 3-7 天 | 找独特角度，差异化切入 |
| 趋势热点 | 1-4 周 | 深度分析，系列化内容 |
| 周期热点 | 可预测 | 提前准备，节日/节气/大事件 |

## 追热内容模板

### 观点输出型
> 关于{热点}，我作为{身份}想说几点...

### 科普解读型
> {热点}到底是怎么回事？一文讲清楚

### 关联延伸型
> 从{热点}聊聊{你的领域}的{相关话题}

### 盘点总结型
> {热点}背后的{N}个真相/启示/教训

## 输出格式

```
🔥 热点分析报告

📌 热点：{热点名称}
📊 热度：⭐⭐⭐⭐⭐
⏰ 预估生命周期：{时间}
⚠️ 风险评估：{低/中/高}

🎯 与你的关联度：{评分}/10
💡 推荐切入角度：
  1. {角度1} — 适合{平台}
  2. {角度2} — 适合{平台}
  3. {角度3} — 适合{平台}

📝 内容方案：
  标题：{建议标题}
  核心观点：{1-2句}
  内容大纲：{简要大纲}
```

## 最佳实践

- **速度第一**：追热点最重要的是速度，完成比完美重要
- **安全第一**：涉及政治、灾难、争议人物的热点谨慎追踪
- **角度为王**：同一热点，独特角度比速度更重要
- **不强蹭**：与账号定位无关的热点不要硬蹭
