## 热点追踪方法论

### 热点生命周期

```
发酵期 → 爆发期 → 高峰期 → 衰退期 → 消亡期
  ↓        ↓        ↓        ↓        ↓
 提前布局  快速响应  深度解读  总结复盘  不再追踪
```

### 追热时机选择

| 阶段 | 特点 | 策略 |
|------|------|------|
| 发酵期 | 热度上升中 | 抢先发布，吃第一波流量 |
| 爆发期 | 热度最高 | 快速响应，蹭流量高峰 |
| 高峰期 | 热度稳定 | 深度解读，差异化内容 |
| 衰退期 | 热度下降 | 总结复盘，长尾流量 |

## 追热风险评估

### 风险类型

| 风险 | 说明 | 应对 |
|------|------|------|
| 舆论反转 | 事件出现反转 | 等待事实明确再追 |
| 敏感话题 | 涉及政治、宗教等 | 避免追踪 |
| 版权问题 | 使用他人素材 | 注意版权 |
| 时效过期 | 热点已过 | 快速响应 |

### 追热决策矩阵

```
              高相关性
                 ↑
    【优先追】  │  【可以追】
    热度高+相关 │  热度低+相关
    ←───────────┼───────────→ 高热度
    【谨慎追】  │  【不追】
    热度高+不相关│  热度低+不相关
                 ↓
              低相关性
```

## 追热内容策略

### 内容角度选择

| 角度 | 适用场景 | 示例 |
|------|----------|------|
| 科普解读 | 专业领域热点 | "XX事件的专业解读" |
| 观点输出 | 争议性话题 | "关于XX，我的看法是..." |
| 经验分享 | 相关经历 | "作为XX，我想说..." |
| 资源整合 | 信息类热点 | "XX相关资源汇总" |
| 幽默调侃 | 轻松话题 | "XX的正确打开方式" |

### 追热注意事项

1. **确认事实**：不传播未经证实的信息
2. **尊重当事人**：不消费他人痛苦
3. **保持客观**：不带节奏、不煽动情绪
4. **注意时效**：热点过了就不要追了
5. **保持调性**：追热也要符合账号定位
