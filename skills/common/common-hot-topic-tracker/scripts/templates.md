# 热点追踪模板

## 热点监控模板

```markdown
# 热点监控日报

## 日期：{YYYY-MM-DD}

## 今日热点

### 热点1
- **话题**：{话题名称}
- **热度**：{高/中/低}
- **来源**：{微博/知乎/抖音}
- **持续时间**：{预估}
- **相关性**：{与账号定位的相关度}
- **追热建议**：{是否追/如何追}

### 热点2
...

## 可追热点筛选

| 热点 | 热度 | 相关性 | 时效性 | 建议 |
|------|------|--------|--------|------|
| {热点1} | ⭐⭐⭐ | ⭐⭐⭐ | 24h | 立即追 |
| {热点2} | ⭐⭐ | ⭐⭐⭐⭐ | 48h | 可以追 |

## 追热内容规划

| 热点 | 内容角度 | 平台 | 预计发布时间 |
|------|----------|------|--------------|
| {热点} | {角度} | {平台} | {时间} |
```

## 热点分析模板

```markdown
# 热点深度分析

## 热点概述
- **话题**：{话题}
- **起源**：{事件起源}
- **发酵过程**：{传播路径}

## 热度分析
- **当前热度**：{数据}
- **峰值预测**：{时间}
- **衰减预测**：{时间}

## 舆论分析
- **主流观点**：{观点}
- **争议点**：{争议}
- **情感倾向**：{正面/负面/中性}

## 追热策略
- **切入角度**：{角度}
- **内容形式**：{形式}
- **发布时机**：{时间}
- **风险评估**：{风险}
```

## 追热内容模板

### 观点输出型

```
【标题】关于{热点}，我有{数字}点想说

【正文】
最近{热点}火了，作为{身份}，我想分享一下我的看法。

第一，{观点1}
{论述}

第二，{观点2}
{论述}

第三，{观点3}
{论述}

你怎么看？欢迎评论区讨论。
```

### 科普解读型

```
【标题】{热点}到底是怎么回事？一文讲清楚

【正文】
最近{热点}刷屏了，很多人还不太了解是怎么回事。

今天我来给大家科普一下：

1. {背景介绍}
2. {事件经过}
3. {影响分析}
4. {未来展望}

希望对大家有帮助。
```
