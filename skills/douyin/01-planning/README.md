# 抖音 - 策划阶段 Skills

策划阶段决定视频的选题方向和内容定位。

## 本阶段目标

- 🔥 发现热门话题，抓住流量机会
- 👥 分析目标用户，明确内容定位
- 📋 规划视频选题，建立内容矩阵
- 🎵 追踪热门音乐，把握流量趋势

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [topic-finder](./topic-finder/) | 热门选题发现，追踪平台热点趋势 | ✅ 可用 |
| bgm-tracker | 热门音乐追踪，发现流量密码 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 topic-finder 发现热门选题
2. 使用 common/hot-topic-tracker 追踪全网热点
3. 使用 common/audience-analyzer 分析目标用户
4. 使用 common/content-calendar 规划发布节奏
```

1. **抖音热榜**：实时热点话题
2. **热门音乐**：BGM 带动的内容趋势
3. **挑战赛**：官方和民间挑战
4. **评论区**：用户需求和问题
