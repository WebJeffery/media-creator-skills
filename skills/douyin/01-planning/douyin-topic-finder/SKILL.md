# 抖音热门选题发现助手

帮助创作者发现和分析抖音热门选题，追踪平台趋势，找到适合自己的内容方向。

## 适用场景

- 寻找当前热门选题方向
- 分析特定赛道的内容趋势
- 规划短视频内容日历
- 发现蓝海选题机会

## 使用方法

```
我是做美食类账号的，帮我找最近的热门选题
```

```
分析一下"知识分享"赛道最近什么选题火
```

## 选题发现框架

### 1. 热点追踪法
- **平台热搜**：抖音热搜榜、挑战赛
- **节日节点**：节假日、纪念日、24 节气
- **社会热点**：新闻事件、行业动态
- **影视综艺**：热播剧、综艺梗

### 2. 竞品分析法
- 分析同赛道头部账号近期爆款
- 关注点赞/评论比异常高的内容
- 研究新晋黑马账号的选题策略

### 3. 评论区挖掘法
- 从自己视频评论区找高频问题
- 从竞品评论区找未被满足的需求
- 从热门视频评论区找争议话题

### 4. 搜索趋势法
- 抖音搜索框的联想词
- 搜索结果中的"相关搜索"
- 关注搜索量上升的关键词

## 选题评估矩阵

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 热度 | 30% | 话题浏览量、搜索趋势 |
| 竞争度 | 25% | 同类内容数量、头部占比 |
| 匹配度 | 25% | 与账号定位的契合程度 |
| 变现力 | 20% | 商业化潜力、广告主需求 |

## 输出格式

```
🔥 热门选题推荐（{赛道}）

1️⃣ {选题名称}
   热度：⭐⭐⭐⭐⭐ | 竞争：⭐⭐⭐ | 推荐指数：9/10
   切入角度：{建议的差异化角度}
   参考账号：{可参考的账号}
   预估数据：{预期播放量级}

2️⃣ {选题名称}
   ...
```

## 最佳实践

- **差异化切入**：热门选题 + 独特角度 = 爆款公式
- **快速响应**：热点选题的时效性很强，24 小时内发布最佳
- **系列化**：将爆款选题做成系列，持续吃透流量
- **数据验证**：先发一条测试，数据好再批量产出
