# 抖音选题发现模板

## 选题来源渠道

### 1. 平台内部

```
【抖音热榜】
- 路径：首页 → 搜索 → 热榜
- 更新：实时更新
- 用途：追热点、蹭流量

【热门音乐】
- 路径：创作 → 选音乐 → 热歌榜
- 更新：每日更新
- 用途：发现内容趋势

【同城热门】
- 路径：首页 → 同城
- 更新：实时
- 用途：本地化内容

【挑战赛】
- 路径：搜索 → 挑战
- 更新：不定期
- 用途：参与官方活动
```

### 2. 外部渠道

| 渠道 | 网址 | 用途 |
|------|------|------|
| 微博热搜 | weibo.com | 社会热点 |
| 知乎热榜 | zhihu.com/hot | 深度话题 |
| 百度指数 | index.baidu.com | 搜索趋势 |

## 选题评估模板

```markdown
# 选题评估表

## 基本信息
- 选题：{选题名称}
- 类型：{口播/剧情/教程/种草}
- 时效性：{热点/常青}

## 评估维度（满分5分）

| 维度 | 评分 | 说明 |
|------|------|------|
| 受众匹配 | /5 | 是否符合目标用户 |
| 完播潜力 | /5 | 能否吸引看完 |
| 互动潜力 | /5 | 能否引发评论 |
| 可执行性 | /5 | 能否高质量完成 |
| 差异化 | /5 | 是否有独特角度 |

## 总分：/25

## 决策
- [ ] 立即执行
- [ ] 加入备选
- [ ] 放弃
```

## 热门选题方向

### 知识科普类

```
- 职场技能：Excel/PPT/沟通技巧
- 生活常识：健康/理财/法律
- 行业揭秘：内幕/真相/避坑
- 学习方法：记忆/阅读/时间管理
```

### 剧情搞笑类

```
- 职场日常：加班/开会/同事关系
- 情感故事：恋爱/友情/家庭
- 社会现象：消费/社交/代际
- 反转段子：误会/巧合/意外
```

### 生活记录类

```
- Vlog日常：一天/一周/一月
- 挑战记录：30天/100天
- 变化记录：减肥/学习/成长
- 体验记录：探店/旅行/尝试
```
