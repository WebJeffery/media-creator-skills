# 抖音 - 创作阶段 Skills

创作阶段是产出高质量短视频内容的核心环节。

## 本阶段目标

- 🎬 撰写视频脚本，规划拍摄内容
- 💡 设计黄金开头，提升完播率
- 📝 编写口播文案，确保表达流畅
- 🎥 规划分镜画面，指导拍摄执行

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [script-writer](./script-writer/) | 短视频脚本撰写，含分镜和口播稿 | ✅ 可用 |
| [hook-generator](./hook-generator/) | 黄金3秒开头生成，提升完播率 | ✅ 可用 |
| storyboard-designer | 分镜脚本设计，可视化拍摄计划 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 hook-generator 设计黄金3秒开头
2. 使用 script-writer 撰写完整脚本
3. 使用 common/copywriting-formulas 优化文案
4. 准备拍摄，按脚本执行
```

## 抖音脚本要点

### 视频结构黄金法则

```
┌─────────────────────────────────────┐
│  钩子（0-3秒）- 决定是否继续观看      │
│  • 悬念 / 冲突 / 利益点 / 视觉冲击   │
├─────────────────────────────────────┤
│  铺垫（3-8秒）- 建立期待             │
│  • 背景交代 / 问题引出 / 承诺价值    │
├─────────────────────────────────────┤
│  核心（8-X秒）- 价值输出             │
│  • 信息传递 / 剧情发展 / 干货分享    │
├─────────────────────────────────────┤
│  高潮（倒数5秒）- 情绪高点           │
│  • 反转 / 金句 / 情感爆发            │
├─────────────────────────────────────┤
│  行动（最后2秒）- 引导互动           │
│  • 点赞 / 关注 / 评论引导            │
└─────────────────────────────────────┘
```

### 不同类型视频脚本

| 类型 | 时长建议 | 核心要素 |
|------|----------|----------|
| 口播干货 | 15-60秒 | 信息密度高、节奏快、字幕清晰 |
| 剧情段子 | 30-90秒 | 人设鲜明、冲突明确、反转有力 |
| 产品种草 | 20-45秒 | 痛点切入、效果展示、真实体验 |
| 教程演示 | 30-120秒 | 步骤清晰、画面对应、可操作 |
| Vlog记录 | 60-180秒 | 节奏紧凑、亮点突出、情感真实 |

### 黄金3秒开头公式

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | 你知道...吗？ | 你知道为什么你总是存不下钱吗？ |
| 利益型 | 学会这个，你就能... | 学会这3招，你的PPT立刻高级 |
| 冲突型 | 千万不要... | 千万不要这样洗脸，会烂脸！ |
| 数字型 | X个/X秒/X步 | 3个动作，告别圆肩驼背 |
| 反问型 | 为什么...？ | 为什么你的视频没人看？ |
| 共鸣型 | 是不是经常...？ | 是不是经常加班到很晚？ |
