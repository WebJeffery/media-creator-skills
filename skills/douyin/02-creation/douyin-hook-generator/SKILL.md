# 抖音黄金3秒开头生成器

为抖音短视频生成高留存率的开头文案，提升完播率和推荐量。

## 适用场景

- 为短视频设计吸引人的开头
- 优化已有视频的开头话术
- 学习不同类型的开头套路

## 使用方法

```
帮我写5个抖音视频开头，主题是"租房避坑指南"
```

```
我要拍一个美食教程视频，帮我设计黄金3秒开头
```

## 开头公式

### 1. 利益前置型
> 「学会这{数字}招，{获得收益}」
> 示例：「学会这3招，每月多省2000块」

### 2. 痛点扎心型
> 「你是不是也{痛点描述}？」
> 示例：「你是不是也觉得工资永远不够花？」

### 3. 反常识型
> 「{常见做法}？你一直都做错了！」
> 示例：「每天喝8杯水？你一直都做错了！」

### 4. 悬念型
> 「{某件事}的结果，我自己都没想到...」
> 示例：「坚持早起100天的结果，我自己都没想到...」

### 5. 权威型
> 「做了{年数}年{职业}，告诉你一个真相」
> 示例：「做了10年HR，告诉你面试的一个真相」

### 6. 紧迫型
> 「{时间限制}之前，一定要{做某事}！」
> 示例：「2025年之前，一定要学会这个技能！」

### 7. 数据冲击型
> 「{惊人数据}，{解释/反转}」
> 示例：「90%的人不知道，手机这个功能可以省一半电」

### 8. 故事开场型
> 「上次{发生了什么}，{意外结果}」
> 示例：「上次去面试迟到了30分钟，结果反而被录用了」

## 输出格式

每个开头包含：
1. 口播文案（3 秒内说完，约 15-20 字）
2. 画面建议（配合什么画面）
3. 字幕文案（屏幕上显示的文字）
4. 公式类型
5. 预估留存效果（1-10）

## 平台规则

- 前 3 秒是算法判断视频质量的关键窗口
- 避免"大家好，我是XXX"等低效开头
- 不得使用虚假信息吸引点击

## 最佳实践

- **直入主题**：不要寒暄，第一句话就给信息
- **制造冲突**：认知冲突、情感冲突、利益冲突
- **语速偏快**：开头语速比正文快 10-20%，制造紧迫感
- **画面配合**：开头画面要有视觉冲击力或变化
- **字幕加持**：大字幕强化开头信息
