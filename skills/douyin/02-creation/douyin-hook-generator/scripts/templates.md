# 抖音黄金开头模板

## 开头公式库

### 1. 悬念型

```
【公式】
你知道{认知}吗？
{现象}，真相竟然是...
千万不要{行为}，否则...

【示例】
- 你知道为什么你的视频没人看吗？
- 这个行业正在消失，真相竟然是...
- 千万不要这样洗脸，否则会烂脸！
```

### 2. 利益型

```
【公式】
学会这{数字}招，你就能{结果}
{时间}教你{技能}
用了这个方法，{效果}

【示例】
- 学会这3招，你的PPT立刻高级
- 30秒教你做出高级感照片
- 用了这个方法，一个月瘦了10斤
```

### 3. 冲突型

```
【公式】
别再{错误做法}了！
停！你的方法完全错了
{数字}%的人都做错了

【示例】
- 别再这样护肤了，越护越差！
- 停！你的简历写法完全错了
- 90%的人都做错了这件事
```

### 4. 共鸣型

```
【公式】
是不是经常{痛点}？
作为一个{身份}，我太懂了
如果你也{情况}，一定要看完

【示例】
- 是不是经常加班到很晚？
- 作为一个打工人，我太懂这种感觉了
- 如果你也经常失眠，一定要看完
```

### 5. 数字型

```
【公式】
{数字}个{方法}，{效果}
从{数字A}到{数字B}，我只用了{方法}
坚持{时间}，{变化}

【示例】
- 3个动作，告别圆肩驼背
- 从月薪3000到30000，我只用了这个方法
- 坚持30天早起，我的变化惊人
```

## 开头节奏模板

```
【0-1秒】视觉冲击/声音吸引
- 突然出现的画面
- 有力的第一句话
- 特殊音效

【1-2秒】抛出钩子
- 悬念/问题/利益点
- 让观众产生好奇

【2-3秒】承诺价值
- 告诉观众看完能得到什么
- 建立继续观看的动机
```
