# 抖音短视频脚本撰写助手

帮助创作者撰写完整的短视频脚本，包含分镜设计、口播稿、字幕文案和拍摄指导。

## 适用场景

- 撰写口播类短视频脚本
- 设计剧情/段子类视频分镜
- 编写产品种草/测评视频脚本
- 制作教程/知识分享类视频脚本

## 使用方法

```
帮我写一个30秒的抖音脚本，主题是"3个Excel快捷键"，口播类型
```

```
写一个1分钟的抖音剧情脚本，主题是"程序员面试的搞笑经历"
```

## 视频类型与时长建议

| 类型 | 推荐时长 | 说明 |
|------|----------|------|
| 口播干货 | 15-60 秒 | 知识分享、技巧教学 |
| 剧情段子 | 30-90 秒 | 搞笑、反转、情感 |
| 产品种草 | 20-45 秒 | 产品展示、使用体验 |
| Vlog 记录 | 60-180 秒 | 生活记录、旅行分享 |
| 教程演示 | 30-120 秒 | 步骤教学、操作演示 |

## 脚本结构

### 黄金结构：钩子 → 内容 → 行动

1. **钩子（0-3秒）**：抓住注意力，阻止划走
2. **铺垫（3-8秒）**：建立期待，引出主题
3. **核心内容（8-X秒）**：价值输出，信息传递
4. **高潮/反转（倒数5秒）**：情绪高点或认知反转
5. **行动引导（最后2秒）**：引导点赞/关注/评论

## 平台规则

- 视频时长：最短 5 秒，最长 15 分钟（推荐 15-60 秒）
- 视频比例：9:16 竖屏为主
- 分辨率：1080×1920 最佳
- 避免搬运、抄袭他人内容
- 禁止低俗、暴力、虚假内容
- 不得出现竞品平台水印

## 最佳实践

- **前 3 秒决定生死**：用悬念、冲突、利益点开头
- **节奏紧凑**：每 3-5 秒一个信息点，避免拖沓
- **字幕必加**：很多用户静音观看，字幕是必需品
- **口播语速**：每分钟 200-250 字为舒适语速
- **完播率优先**：短视频的核心指标是完播率，宁短勿长
