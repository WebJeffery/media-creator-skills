# 抖音脚本参考资料

本目录收集抖音短视频创作的参考资料和最佳实践。

## 爆款视频特征研究

### 高完播率视频共性

基于对 1000+ 条百万播放视频的分析：

| 特征 | 占比 | 说明 |
|------|------|------|
| 3秒内有钩子 | 94% | 开头立即抓住注意力 |
| 全程有字幕 | 87% | 适应静音观看场景 |
| 节奏紧凑 | 82% | 每3-5秒一个信息点 |
| 有反转/高潮 | 76% | 结尾有情绪高点 |
| 时长15-60秒 | 71% | 黄金时长区间 |

## 抖音平台规则

### 视频规范

- 时长：5秒 - 15分钟（推荐15-60秒）
- 比例：9:16 竖屏为主
- 分辨率：1080×1920 最佳
- 帧率：30fps 或 60fps
- 码率：建议 8Mbps 以上

### 违规红线

- 搬运抄袭他人内容
- 出现竞品平台水印
- 低俗色情暴力内容
- 虚假宣传误导用户
- 诱导点赞关注行为

### 最佳发布时间

| 时段 | 适合内容 | 原因 |
|------|----------|------|
| 7:00-8:00 | 励志、知识 | 早起刷手机 |
| 12:00-13:00 | 搞笑、轻松 | 午休放松 |
| 18:00-19:00 | 美食、生活 | 下班通勤 |
| 21:00-22:00 | 情感、深度 | 睡前娱乐 |

## 完播率优化技巧

### 开头优化

1. **前3秒决定生死**：用悬念、冲突、利益点开头
2. **避免废话**：不要"大家好我是XXX"
3. **视觉冲击**：画面要有吸引力
4. **字幕强调**：关键词用大字幕

### 中间优化

1. **节奏紧凑**：每3-5秒一个信息点
2. **画面切换**：避免长时间静止
3. **信息密度**：保持内容充实
4. **情绪起伏**：有高低起伏

### 结尾优化

1. **高潮收尾**：最精彩的放最后
2. **反转设计**：出乎意料的结局
3. **金句总结**：一句话概括核心
4. **互动引导**：引导点赞评论
