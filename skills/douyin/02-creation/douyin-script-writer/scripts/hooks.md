## 使用技巧

1. **语速要快**：前3秒信息密度要高，语速比正常快20%
2. **表情要夸张**：面部表情要有感染力
3. **字幕要大**：关键词用大字幕强调
4. **画面要动**：避免静止画面，保持视觉吸引
5. **音乐要配合**：BGM 在开头要有记忆点
