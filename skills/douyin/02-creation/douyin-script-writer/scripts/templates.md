# 抖音脚本结构模板

## 口播干货脚本模板

```
🎬 视频信息
- 类型：口播干货
- 时长：30-60秒
- 目标：涨粉/互动

📝 脚本内容

【钩子 0-3秒】
"{悬念/利益点开头}"
画面：{画面描述}
字幕：{字幕内容}

【铺垫 3-8秒】
"{问题引出/背景交代}"
画面：{画面描述}
字幕：{字幕内容}

【核心内容 8-25秒】
"第一点，{要点1}
第二点，{要点2}
第三点，{要点3}"
画面：{配合画面/图片}
字幕：{分点字幕}

【总结 25-28秒】
"{核心观点总结}"
画面：{画面描述}
字幕：{字幕内容}

【行动引导 28-30秒】
"觉得有用的话，点个赞收藏一下，关注我学习更多{领域}知识"
画面：{指向点赞按钮}
字幕：点赞+关注

🎵 BGM建议：{轻快/励志/无BGM}
```

## 剧情段子脚本模板

```
🎬 视频信息
- 类型：剧情段子
- 时长：45-90秒
- 目标：播放量/互动

📝 脚本内容

【开场 0-5秒】
场景：{场景描述}
人物：{人物介绍}
台词："{开场台词}"
画面：{镜头描述}

【铺垫 5-20秒】
场景：{场景描述}
台词：
A："{台词}"
B："{台词}"
画面：{镜头切换}

【冲突 20-35秒】
场景：{冲突场景}
台词：
A："{台词}"
B："{台词}"
画面：{特写/切换}
BGM：{音乐变化}

【高潮/反转 35-45秒】
场景：{反转场景}
台词："{反转台词}"
画面：{慢动作/特效}
BGM��{高潮音乐}

【结尾 45-50秒】
场景：{收尾场景}
台词："{结尾台词/金句}"
画面：{定格/字幕}

🎵 BGM建议：{搞笑/反转音效}
```

## 产品种草脚本模板

```
🎬 视频信息
- 类型：产品种草
- 时长：20-45秒
- 目标：转化/互动

📝 脚本内容

【痛点切入 0-5秒】
"{目标用户的痛点问题}"
画面：{痛点场景展示}
字幕：{痛点文字}

【产品引入 5-10秒】
"直到我发现了这个{产品名称}"
画面：{产品展示}
字幕：{产品名称}

【效果展示 10-25秒】
"用了之后，{效果描述}"
画面：{使用过程/效果对比}
字幕：{效果要点}

【使用方法 25-35秒】
"使用方法很简单，{步骤说明}"
画面：{操作演示}
字幕：{步骤文字}

【总结推荐 35-40秒】
"如果你也有{痛点}，真的可以试试"
画面：{产品特写}
字幕：{推荐语}

【行动引导 40-45秒】
"链接放在{位置}，需要的自取"
画面：{指向链接}
字幕：点击购买

🎵 BGM建议：{轻快/种草类热门BGM}
```

## 教程演示脚本模板

```
🎬 视频信息
- 类型：教程演示
- 时长：30-120秒
- 目标：收藏/涨粉

📝 脚本内容

【效果展示 0-5秒】
"今天教大家{技能}，效果是这样的"
画面：{最终效果展示}
字幕：{效果描述}

【步骤讲解】

第一步（5-15秒）
"{步骤1说明}"
画面：{操作演示}
字幕：Step 1: {步骤名称}

第二步（15-25秒）
"{步骤2说明}"
画面：{操作演示}
字幕：Step 2: {步骤名称}

第三步（25-35秒）
"{步骤3说明}"
画面：{操作演示}
字幕：Step 3: {步骤名称}

【注意事项 35-40秒】
"这里要注意，{注意点}"
画面：{重点标注}
字幕：⚠️ 注意

【效果对比 40-45秒】
"来看一下前后对比"
画面：{对比展示}
字幕：Before → After

【结尾引导 45-50秒】
"学会了吗？收藏起来慢慢学"
画面：{点赞收藏引导}
字幕：收藏+关注

🎵 BGM建议：{轻快/教程类热门BGM}
```
