# 抖音 - 优化阶段 Skills

优化阶段决定视频的曝光量和互动效果。

## 本阶段目标

- 📝 生成吸睛标题，提升点击率
- 🔍 优化搜索排名，增加曝光机会
- 🏷️ 选择热门话题，蹭流量红利
- 🎵 匹配热门音乐，提升推荐权重

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [title-generator](./title-generator/) | 抖音标题和文案生成，含热门话题适配 | ✅ 可用 |
| [seo-optimizer](./seo-optimizer/) | 抖音 SEO 优化，提升搜索排名 | ✅ 可用 |
| tag-optimizer | 话题标签优化，提升视频曝光 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 title-generator 生成视频标题和文案
2. 使用 seo-optimizer 优化关键词布局
3. 选择合适的话题标签（3-5个）
4. 匹配热门 BGM，最终发布
```

## 抖音优化要点

### 标题文案优化

| 要素 | 建议 | 示例 |
|------|------|------|
| 字数 | 建议100字以内 | - |
| 关键词 | 前置核心关键词 | 「减肥」放在开头 |
| 话题 | 3-5个相关话题 | #减肥 #健身 #瘦身 |
| 互动 | 引导评论互动 | 你们觉得呢？ |
| @账号 | 可@相关账号 | @抖音小助手 |

### SEO 优化策略

| 优化点 | 说明 |
|--------|------|
| 标题关键词 | 在标题中包含目标搜索词 |
| 文案关键词 | 在描述中自然融入关键词 |
| 话题标签 | 选择搜索量大的话题 |
| 口播关键词 | 视频中口播目标关键词 |
| 字幕关键词 | 字幕中包含关键词 |

### 话题标签策略

| 标签类型 | 数量 | 示例 |
|----------|------|------|
| 核心标签 | 1-2个 | #减肥 #健身 |
| 细分标签 | 1-2个 | #居家健身 #燃脂运动 |
| 热门标签 | 1个 | #抖音小助手 |

### 发布前检查清单

- [ ] 视频画质是否清晰（1080×1920）
- [ ] 视频时长是否合适（建议15-60秒）
- [ ] 开头3秒是否足够吸引人
- [ ] 字幕是否清晰完整
- [ ] 标题是否包含关键词
- [ ] 话题标签是否选择合适
- [ ] BGM 是否匹配内容
- [ ] 封面是否吸引人
- [ ] 发布时间是否最佳（12:00-13:00/18:00-20:00）
- [ ] 是否有违规内容或水印
