# 抖音 SEO 优化助手

帮助创作者优化短视频的搜索排名，通过关键词布局提升视频在抖音搜索中的曝光量。

## 适用场景

- 优化视频标题和描述的关键词布局
- 分析特定关键词的搜索竞争度
- 制定抖音搜索流量获取策略
- 优化账号整体的搜索表现

## 使用方法

```
帮我优化这个抖音视频的SEO，主题是"Python入门教程"
```

```
分析"家常菜做法"这个关键词在抖音的搜索情况
```

## 抖音 SEO 关键要素

### 1. 关键词布局位置（按权重排序）

| 位置 | 权重 | 说明 |
|------|------|------|
| 视频口播内容 | ⭐⭐⭐⭐⭐ | 抖音会识别语音中的关键词 |
| 视频标题 | ⭐⭐⭐⭐⭐ | 搜索匹配的核心位置 |
| 字幕文字 | ⭐⭐⭐⭐ | OCR 识别字幕中的关键词 |
| 话题标签 | ⭐⭐⭐⭐ | 话题关联搜索结果 |
| 视频描述 | ⭐⭐⭐ | 补充关键词的位置 |
| 评论区 | ⭐⭐⭐ | 评论中的关键词也会被索引 |
| 账号昵称/简介 | ⭐⭐ | 影响账号级别的搜索排名 |

### 2. 关键词选择策略

- **核心词**：搜索量大，竞争激烈（如"减肥"）
- **长尾词**：搜索量中等，竞争较小（如"学生党减肥食谱"）
- **问题词**：用户搜索的问题（如"怎么快速减肥"）
- **场景词**：特定场景关键词（如"办公室减肥操"）

### 3. 关键词密度建议

- 标题中出现 1-2 次核心关键词
- 口播中自然提及 3-5 次
- 字幕中完整展示关键词
- 描述中包含 1-2 个长尾关键词

## 输出格式

```
🔍 SEO 优化方案

📌 核心关键词：{关键词}
📌 长尾关键词：{关键词1}、{关键词2}、{关键词3}

✅ 标题优化：
   原标题：{原标题}
   优化后：{优化标题}

✅ 描述优化：
   {优化后的描述文案}

✅ 话题标签：
   {推荐话题标签}

✅ 口播关键词植入建议：
   - 开头（0-3s）：提及"{关键词}"
   - 中间：自然融入"{长尾词}"
   - 结尾：再次强调"{关键词}"

✅ 字幕优化建议：
   {字幕中需要突出的关键词}
```

## 平台规则

- 关键词需与视频内容高度相关
- 避免关键词堆砌（会被判定为低质内容）
- 不得使用误导性关键词

## 最佳实践

- **搜索框联想**：在抖音搜索框输入关键词，查看联想词作为长尾词
- **自然植入**：关键词要自然融入内容，不要生硬堆砌
- **持续优化**：定期分析搜索数据，调整关键词策略
- **矩阵布局**：围绕核心关键词制作系列视频，占据搜索结果
