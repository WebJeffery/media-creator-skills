# 抖音SEO优化模板

## 关键词布局模板

### 关键词位置

```
【标题】
{核心关键词} + {描述}
示例：Excel技巧｜3个快捷键提升10倍效率

【文案】
{关键词}相关的描述文案
示例：学会这3个Excel快捷键，工作效率翻倍

【口播】
视频中口播{关键词}
示例："今天教大家3个Excel快捷键..."

【字幕】
字幕中包含{关键词}
示例：字幕显示"Excel快捷键"

【话题】
#{关键词} #{相关词}
示例：#Excel #Excel技巧 #办公技巧
```

### 关键词研究模板

```markdown
# 关键词研究

## 核心关键词
- 主关键词：{词}
- 搜索量：{高/中/低}
- 竞争度：{高/中/低}

## 长尾关键词
1. {长尾词1}
2. {长尾词2}
3. {长尾词3}

## 相关关键词
1. {相关词1}
2. {相关词2}
3. {相关词3}

## 关键词布局计划
- 标题：{使用的关键词}
- 文案：{使用的关键词}
- 话题：{使用的关键词}
```

## SEO检查清单

```
【发布前检查】
- [ ] 标题是否包含核心关键词
- [ ] 关键词是否在标题前半部分
- [ ] 文案是否自然融入关键词
- [ ] 话题标签是否包含关键词
- [ ] 视频口播是否提及关键词
- [ ] 字幕是否包含关键词

【发布后追踪】
- [ ] 搜索排名位置
- [ ] 搜索流量占比
- [ ] 关键词点击率
```
