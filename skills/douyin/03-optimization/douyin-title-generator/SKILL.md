# 抖音标题和文案生成器

为抖音短视频生成标题和发布文案，含热门话题适配，提升视频搜索曝光和互动率。

## 适用场景

- 为短视频生成发布标题和描述文案
- 优化已有视频的标题和文案
- 生成适合抖音 SEO 的关键词文案

## 使用方法

```
帮我写抖音发布文案，视频内容是"教你3分钟做一道番茄炒蛋"
```

```
生成5个抖音标题，主题是"职场新人必备技能"
```

## 标题公式

### 1. 疑问互动型
> 「{问题}？评论区告诉我」
> 提升评论量，触发算法推荐

### 2. 争议观点型
> 「我觉得{争议观点}，你们觉得呢？」
> 引发讨论，提升互动率

### 3. 干货价值型
> 「{数字}个{技巧}，第{N}个太实用了」
> 提供价值感，提升收藏量

### 4. 情感共鸣型
> 「{情感场景}，看完泪目了...」
> 触发情感共鸣，提升转发量

### 5. @特定人群型
> 「@{人群} 这条视频就是为你拍的」
> 精准触达目标用户

## 文案结构

```
{核心标题}

{1-2句补充说明或金句}

{互动引导：提问/投票/挑战}

{话题标签 3-5个}
```

## 输出格式

1. **标题**（核心标题，30 字以内）
2. **描述文案**（含互动引导，100 字以内）
3. **话题标签**（3-5 个热门话题）
4. **互动引导语**（引导评论/点赞的话术）

## 平台规则

- 标题/描述总字数建议 100 字以内
- 话题标签 3-5 个为佳
- 避免引导私信、加微信等导流行为
- 禁止虚假标题党

## 最佳实践

- **关键词前置**：核心关键词放在文案开头，利于搜索
- **引导互动**：用提问句结尾，提升评论量
- **话题蹭热**：适当关联热门话题，但需与内容相关
- **留悬念**：文案不说完，引导用户看视频
