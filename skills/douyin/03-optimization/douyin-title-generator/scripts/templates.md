# 抖音标题文案模板

## 标题公式库

### 1. 数字+效果型

```
{数字}个{方法}，{效果}
{时间}学会{技能}
从{数字A}到{数字B}的秘密
```

**示例**：
- 3个动作，告别圆肩驼背
- 5分钟学会高级感穿搭
- 从0到10万粉的秘密

### 2. 痛点+方案型

```
{痛点}？试试这个方法
为什么你{痛点}？原因在这
{痛点}的人一定要看
```

**示例**：
- 总是存不下钱？试试这个方法
- 为什么你的视频没人看？原因在这
- 经常失眠的人一定要看

### 3. 悬念+好奇型

```
{现象}，真相竟然是...
你绝对想不到{结果}
{数字}%的人都不知道
```

**示例**：
- 这个行业正在消失，真相竟然是...
- 你绝对想不到这个方法这么有效
- 90%的人都不知道的护肤真相

### 4. 身份+共鸣型

```
{身份}必看！{内容}
作为{身份}，我想说...
{身份}的{场景}日常
```

**示例**：
- 打工人必看！高效摸鱼指南
- 作为过来人，我想说这3点
- 程序员的加班日常

## 文案结构模板

```
【标题】{吸引点击的标题}

【正文】
{开头：痛点/悬念/利益}
{中间：核心内容概述}
{结尾：互动引导}

【话题】
#{核心话题} #{人群话题} #{热门话题}

【@账号】
@抖音小助手（可选）
```

## 互动引导话术

```
【点赞引导】
- 觉得有用的点个赞
- 同意的双击屏幕
- 点赞过100更新下一期

【评论引导】
- 你们觉得呢？评论区聊聊
- 还想看什么内容？评论告诉我
- 你是哪种类型？评论区报数

【关注引导】
- 关注我，学习更多{领域}知识
- 点击头像关注，不错过更新
- 关注+收藏，下次找得到
```
