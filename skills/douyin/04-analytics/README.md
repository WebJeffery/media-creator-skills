# 抖音 - 分析阶段 Skills

分析阶段帮助创作者了解视频表现，持续优化创作策略。

## 本阶段目标

- 📊 追踪视频数据，了解内容表现
- 📈 分析用户行为，优化内容策略
- 🔎 监控竞品动态，学习爆款规律
- 🔮 预测内容趋势，提前布局选题

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| performance-tracker | 视频数据追踪，监控核心指标 | 🔲 规划中 |
| engagement-analyzer | 互动数据分析，了解用户偏好 | 🔲 规划中 |
| competitor-monitor | 竞品账号监控，学习爆款规律 | 🔲 规划中 |

## 抖音核心数据指标

### 视频表现指标

| 指标 | 说明 | 重要性 |
|------|------|--------|
| 完播率 | 完整观看人数/播放人数 | ⭐⭐⭐⭐⭐ |
| 点赞率 | 点赞数/播放量 | ⭐⭐⭐⭐ |
| 评论率 | 评论数/播放量 | ⭐⭐⭐⭐ |
| 转发率 | 转发数/播放量 | ⭐⭐⭐⭐ |
| 关注率 | 新增关注/播放量 | ⭐⭐⭐ |
| 播放量 | 视频被播放的次数 | ⭐⭐⭐ |

### 抖音推荐算法核心

```
┌─────────────────────────────────────────────────┐
│              抖音推荐机制简化模型                │
├─────────────────────────────────────────────────┤
│                                                 │
│  初始流量池（200-500播放）                       │
│        ↓                                        │
│  数据表现评估（完播率 > 点赞 > 评论 > 转发）     │
│        ↓                                        │
│  ┌─────────────┐    ┌─────────────┐            │
│  │ 数据达标    │    │ 数据不达标  │            │
│  │ 进入下一池  │    │ 停止推荐    │            │
│  └─────────────┘    └─────────────┘            │
│        ↓                                        │
│  更大流量池（1000 → 5000 → 10000 → ...）        │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 数据优化方向

| 问题 | 可能原因 | 优化方向 |
|------|----------|----------|
| 完播率低 | 开头不吸引、内容拖沓 | 优化开头、加快节奏 |
| 点赞率低 | 内容价值不足、情感不够 | 提升内容质量、增加情感 |
| 评论率低 | 缺乏互动引导、话题性弱 | 增加互动引导、制造话题 |
| 转发率低 | 实用性不足、社交价值低 | 增加实用内容、提升分享动机 |
| 关注率低 | 人设不清晰、系列感弱 | 强化人设、做系列内容 |
