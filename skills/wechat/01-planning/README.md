# 公众号 - 策划阶段 Skills

策划阶段是内容创作的起点，决定了内容的方向和质量上限。

## 本阶段目标

- 🔥 追踪热点趋势，发现创作机会
- 👥 分析目标受众，明确内容定位
- 📋 规划选题方向，建立内容体系
- 📅 制定发布计划，保持更新节奏

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [topic-planner](./topic-planner/) | 周/月选题规划，生成内容日历 | ✅ 可用 |
| niche-analyzer | 细分领域分析，找到差异化定位 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 common/hot-topic-tracker 追踪全网热点
2. 使用 common/audience-analyzer 分析目标受众
3. 使用 topic-planner 规划周/月选题
4. 使用 common/content-calendar 安排发布日程
```

## 公众号选题要点

### 选题来源

1. **热点追踪**：微博热搜、知乎热榜、行业新闻
2. **用户反馈**：评论区问题、后台留言、社群讨论
3. **竞品分析**：同类账号爆款、行业头部内容
4. **日历节点**：节日、纪念日、行业大事件
5. **系列规划**：主题系列、连载内容、专栏

### 选题评估维度

| 维度 | 说明 | 权重 |
|------|------|------|
| 受众匹配度 | 是否符合目标读者需求 | ⭐⭐⭐⭐⭐ |
| 时效性 | 是否有时间窗口限制 | ⭐⭐⭐⭐ |
| 差异化 | 是否有独特角度或观点 | ⭐⭐⭐⭐ |
| 可执行性 | 素材是否充足、能否高质量完成 | ⭐⭐⭐ |
| 传播潜力 | 是否具备分享动机 | ⭐⭐⭐ |
