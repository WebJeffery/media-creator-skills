# 公众号选题规划助手

帮助公众号运营者进行系统化选题规划，生成周/月内容日历，确保内容持续输出。

## 适用场景

- 制定周/月/季度内容规划
- 围绕账号定位生成选题方向
- 结合热点事件规划追热内容
- 建立选题素材库

## 使用方法

```
我的公众号定位是"职场成长"，帮我规划下周的5篇选题
```

```
帮我做一个月度内容日历，账号方向是"家庭教育"，每周发3篇
```

## 选题框架

### 内容四象限
| | 高传播 | 低传播 |
|---|---|---|
| **高价值** | 爆款干货（优先） | 深度长文（品牌） |
| **低价值** | 热点追踪（流量） | 日常互动（维护） |

### 选题来源
1. **读者痛点**：评论区高频问题、私信咨询
2. **行业热点**：行业新闻、政策变化、技术趋势
3. **节日节点**：节假日、纪念日、行业大会
4. **竞品分析**：同类账号的爆款选题
5. **经典常青**：不受时效影响的长期价值内容

## 输出格式

```
📅 第X周内容日历（MM.DD - MM.DD）

周一 | [选题标题] | 类型：干货文 | 预估阅读：⭐⭐⭐
     | 核心观点：...
     | 目标读者：...

周三 | [选题标题] | 类型：观点文 | 预估阅读：⭐⭐⭐⭐
     | 核心观点：...
     | 目标读者：...

周五 | [选题标题] | 类型：清单文 | 预估阅读：⭐⭐⭐
     | 核心观点：...
     | 目标读者：...
```

## 最佳实践

- **721 法则**：70% 垂直领域内容 + 20% 相关延伸 + 10% 热点追踪
- **系列化**：将大主题拆分为系列文章，培养读者期待
- **节奏感**：干货文和轻松文交替发布，避免读者疲劳
- **提前储备**：保持 2 周以上的选题储备，避免临时凑数
