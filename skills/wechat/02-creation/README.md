# 公众号 - 创作阶段 Skills

创作阶段是将选题转化为高质量内容的核心环节。

## 本阶段目标

- ✍️ 撰写高质量长文，输出有价值的内容
- 🔄 改写优化内容，提升表达效果
- 📝 构建文章结构，确保逻辑清晰
- 💡 打磨开头结尾，提升阅读体验

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [article-writer](./article-writer/) | 公众号长文写作，支持多种文章类型 | ✅ 可用 |
| [content-rewriter](./content-rewriter/) | 内容改写优化，保持原意提升表达 | ✅ 可用 |
| outline-generator | 文章大纲生成，快速构建结构 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 common/copywriting-formulas 选择文案框架
2. 使用 article-writer 撰写文章初稿
3. 使用 content-rewriter 优化表达和风格
4. 人工审核润色，确保内容质量
```

## 公众号写作要点

### 文章结构黄金法则

```
┌─────────────────────────────────────┐
│  开头（前3行）- 决定是否继续阅读      │
│  • 痛点切入 / 悬念设置 / 数据震撼    │
├─────────────────────────────────────┤
│  正文（分段展开）- 价值输出          │
│  • 每300-500字一个小标题            │
│  • 观点+案例+总结 的节奏            │
│  • 重点内容加粗标注                 │
├─────────────────────────────────────┤
│  结尾（引导互动）- 促进传播          │
│  • 总结核心观点                     │
│  • 引导评论/转发/关注               │
└─────────────────────────────────────┘
```

### 不同文章类型写作要点

| 类型 | 核心要素 | 字数建议 |
|------|----------|----------|
| 干货文 | 实用性、可操作、有案例 | 2000-4000 |
| 观点文 | 独特视角、逻辑严密、有态度 | 1500-3000 |
| 故事文 | 情感共鸣、细节描写、有转折 | 2000-3500 |
| 清单文 | 结构清晰、信息密度高、易收藏 | 1500-2500 |
