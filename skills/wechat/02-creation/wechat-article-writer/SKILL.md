# 公众号长文写作助手

帮助创作者快速撰写高质量的微信公众号文章，支持多种文章类型，自动适配公众号阅读体验。

## 适用场景

- 撰写公众号原创长文（干货分享、观点输出、故事叙述）
- 将素材/大纲扩写为完整文章
- 针对特定主题生成不同风格的文章
- 改编其他平台内容为公众号风格

## 使用方法

**基础用法：**
```
帮我写一篇公众号文章，主题是"远程办公的利与弊"
```

**进阶用法（指定类型和风格）：**
```
写一篇公众号干货文，主题是"新手如何学Python"，风格轻松幽默，目标读者是大学生
```

**从大纲扩写：**
```
根据以下大纲写一篇公众号文章：
1. 引言：AI 正在改变内容创作
2. 三个核心变化
3. 创作者如何应对
4. 总结与展望
```

## 文章类型

| 类型 | 说明 | 字数建议 |
|------|------|----------|
| 干货文 | 知识分享、教程指南 | 2000-4000 字 |
| 观点文 | 热点评论、行业分析 | 1500-3000 字 |
| 故事文 | 案例故事、个人经历 | 2000-3500 字 |
| 清单文 | 工具推荐、方法论清单 | 1500-2500 字 |
| 深度文 | 行业研究、趋势分析 | 3000-6000 字 |

## 输出格式

输出包含以下部分：

1. **标题**（含 1-2 个备选标题）
2. **摘要**（用于公众号摘要栏，35 字以内）
3. **正文**（含小标题分段、重点加粗）
4. **引导语**（文末引导关注/转发）

## 平台规则

- 公众号文章无硬性字数上限，但建议 1500-5000 字为最佳阅读区间
- 标题上限 64 个字符
- 摘要上限 120 个字符
- 正文支持 Markdown 基础格式（加粗、引用、列表等）
- 避免诱导分享的话术（如"不转不是中国人"）
- 避免绝对化用语（如"最好"、"第一"、"100%"）

## 最佳实践

- **开头黄金 3 行**：前 3 行决定读者是否继续阅读，用痛点/悬念/数据开头
- **小标题分段**：每 300-500 字设置一个小标题，降低阅读压力
- **重点加粗**：核心观点和关键数据加粗处理
- **适当留白**：段落间保持空行，提升移动端阅读体验
- **结尾引导**：文末设置互动问题或引导关注
