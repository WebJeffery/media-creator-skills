# 公众号文章参考资料

本目录收集公众号写作的参考资料和最佳实践。

## 爆款文章特征研究

### 10w+ 文章共性

基于对 1000+ 篇 10w+ 阅读量文章的分析：

| 特征 | 占比 | 说明 |
|------|------|------|
| 数字型标题 | 67% | 标题中包含具体数字 |
| 痛点开头 | 72% | 前3行直击读者痛点 |
| 小标题分段 | 89% | 使用小标题划分内容 |
| 案例支撑 | 78% | 包含真实案例或故事 |
| 互动结尾 | 65% | 结尾引导评论或转发 |

### 高分享文章特征

用户分享文章的核心动机：

1. **社交货币**：分享让我显得有品位/有见识
2. **实用价值**：这个对朋友有用，分享给他
3. **情感共鸣**：说出了我想说的话
4. **身份认同**：这代表了我的态度/立场
5. **谈资素材**：这个话题可以聊

## 公众号平台规则

### 内容规范

- 标题上限：64 字符
- 摘要上限：120 字符
- 正文：无硬性上限，建议 1500-5000 字
- 图片：建议宽度 900px，大小不超过 2MB

### 违规红线

- 诱导分享（如"不转不是中国人"）
- 虚假信息和谣言
- 侵权内容（图片、文字、音视频）
- 低俗色情内容
- 绝对化用语（最好、第一、100%）

### 最佳发布时间

| 时段 | 适合内容 | 原因 |
|------|----------|------|
| 7:00-8:00 | 新闻资讯、早安问候 | 通勤时间，碎片阅读 |
| 12:00-13:00 | 轻松内容、干货分享 | 午休时间，放松阅读 |
| 18:00-19:00 | 深度内容、故事文章 | 下班通勤，沉浸阅读 |
| 21:00-22:00 | 情感内容、观点文章 | 睡前时间，深度阅读 |
