# AI 写作提示词模板

## 基础写作提示词

```
你是一位资深的公众号内容创作者，擅长撰写高质量的原创文章。

请根据以下信息撰写一篇公众号文章：

【主题】{主题}
【文章类型】{干货文/观点文/故事文/清单文}
【目标读者】{读者画像}
【字数要求】{字数范围}
【风格要求】{专业严谨/轻松幽默/温暖治愈/犀利深刻}

要求：
1. 开头3行内抓住读者注意力
2. 每300-500字设置一个小标题
3. 核心观点和数据加粗标注
4. 结尾设置互动引导
5. 避免使用绝对化用语
```

## 干货文提示词

```
请撰写一篇公众号干货文章：

【主题】{主题}
【核心价值】读者看完能学到什么/解决什么问题
【目标读者】{读者画像}

结构要求：
1. 开头：用痛点或数据引入，让读者产生"这说的就是我"的感觉
2. 正文：分3-5个要点展开，每个要点包含"是什么-为什么-怎么做"
3. 结尾：总结核心要点，给出立即可执行的第一步

风格：专业但不枯燥，有案例有数据，让读者觉得"干货满满"
```

## 观点文提示词

```
请撰写一篇公众号观点文章：

【话题】{话题}
【你的立场】{支持/反对/中立分析}
【核心论点】{一句话概括你的观点}

结构要求：
1. 开头：引出话题，表明立场
2. 现象描述：客观呈现现状
3. 深度分析：剖析原因，展示思考深度
4. 观点输出：清晰表达你的看法
5. 行动建议：给读者的具体建议

风格：有态度但不偏激，有深度但不晦涩，让读者觉得"说得有道理"
```

## 故事文提示词

```
请撰写一篇公众号故事文章：

【故事主题】{主题}
【故事类型】{个人经历/他人案例/虚构故事}
【核心情感】{励志/感动/反思/治愈}
【想传达的道理】{一句话}

结构要求：
1. 开头：场景描写或悬念设置，让读者想继续看
2. 起：交代背景，建立共情
3. 承：事件发展，制造期待
4. 转：转折高潮，情感爆发
5. 合：结局升华，点明主题

风格：细节丰富，情感真挚，让读者觉得"感同身受"
```

## 改写优化提示词

```
请优化以下公众号文章：

【原文】
{粘贴原文}

优化方向：
1. 开头是否足够吸引人？如果不够，请重写
2. 小标题是否清晰有力？如果不够，请优化
3. 核心观点是否突出？如果不够，请加粗标注
4. 结尾是否有引导？如果没有，请添加
5. 整体是否有冗余？如果有，请精简

请输出优化后的完整文章，并说明主要修改点。
```
