# 公众号文章结构模板

## 干货文模板

```markdown
# {标题}

{开头：痛点/问题/数据引入，3行以内}

## {小标题1：核心概念/背景}

{段落1：解释说明}

{段落2：案例/数据支撑}

## {小标题2：方法/步骤}

**{要点1}**：{具体说明}

**{要点2}**：{具体说明}

**{要点3}**：{具体说明}

## {小标题3：实操建议}

{具体可执行的建议}

## 总结

{核心观点回顾，1-2句话}

---

{引导语：关注/转发/评论引导}
```

## 观点文模板

```markdown
# {标题}

{开头：引出话题，表明立场}

## 现象：{描述现状}

{具体现象描述}

{数据或案例支撑}

## 分析：{深入剖析}

{原因分析1}

{原因分析2}

## 观点：{核心论述}

{你的核心观点}

{论据支撑}

## 建议：{行动指南}

{给读者的具体建议}

---

**你怎么看？欢迎在评论区留言讨论。**
```

## 故事文模板

```markdown
# {标题}

{开头：场景描写/悬念设置}

## {起}

{故事背景，人物介绍}

## {承}

{事件发展，矛盾铺垫}

## {转}

{转折点，高潮部分}

## {合}

{结局，感悟升华}

---

{引导语}
```

## 清单文模板

```markdown
# {数字}个{主题}，{效果描述}

{开头：为什么整理这个清单}

## 1. {清单项1}

- **是什么**：{简要说明}
- **为什么推荐**：{推荐理由}
- **怎么用**：{使用建议}

## 2. {清单项2}

...

## {数字}. {清单项N}

...

## 总结

{清单价值总结}

{收藏/转发引导}
```
