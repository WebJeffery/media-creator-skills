# 公众号内容改写助手

将已有内容改写为公众号风格，保持核心观点不变的同时优化表达方式和阅读体验。

## 适用场景

- 将学术/专业内容改写为通俗易懂的公众号文章
- 将其他平台内容（知乎、微博等）适配为公众号风格
- 对已有文章进行风格转换（严肃→轻松、专业→科普）
- 降低文章重复率，保持原意

## 使用方法

```
将以下内容改写为公众号风格：
[粘贴原文]
```

```
把这篇学术论文摘要改写成公众号科普文，目标读者是普通大众
```

## 改写维度

| 维度 | 说明 |
|------|------|
| 风格转换 | 学术→通俗、严肃→轻松、客观→感性 |
| 结构优化 | 重新组织段落、添加小标题、优化过渡 |
| 表达升级 | 替换书面语为口语化表达、增加比喻和案例 |
| 平台适配 | 适配公众号排版习惯、阅读节奏 |

## 输出格式

1. **改写后的完整文章**
2. **改写说明**（列出主要修改点）
3. **原创度评估**（与原文的差异程度）

## 平台规则

- 改写需保持原文核心观点，不得歪曲原意
- 如引用他人观点需注明出处
- 避免简单的同义词替换，需要真正的结构和表达重组

## 最佳实践

- **理解再改写**：先理解原文核心逻辑，再用自己的方式重新表达
- **增加案例**：用贴近读者生活的案例替换抽象描述
- **口语化**：适当使用"你"、"我们"等拉近距离
- **节奏感**：长短句交替，避免连续长句
