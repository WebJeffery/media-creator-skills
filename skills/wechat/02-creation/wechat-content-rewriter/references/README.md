# 内容改写参考资料

## 改写技巧

### 1. 句式变换

| 原句式 | 变换方式 | 示例 |
|--------|----------|------|
| 主动句 | 改为被动句 | "我完成了任务" → "任务被完成了" |
| 陈述句 | 改为反问句 | "这很重要" → "这难道不重要吗？" |
| 长句 | 拆分短句 | 一句话拆成2-3句 |
| 短句 | 合并长句 | 多个短句合并 |

### 2. 词汇替换

| 原词 | 同义替换 |
|------|----------|
| 重要 | 关键、核心、至关重要、不可或缺 |
| 提高 | 提升、增强、优化、改善 |
| 方法 | 技巧、策略、途径、方式 |
| 问题 | 难题、挑战、困境、痛点 |
| 解决 | 处理、应对、化解、攻克 |

### 3. 结构调整

**时间顺序 → 重要性顺序**
- 原：先发生的先写
- 改：最重要的先写

**总分结构 → 分总结构**
- 原：先总结后分述
- 改：先分述后总结

**并列结构 → 递进结构**
- 原：A、B、C 并列
- 改：A → B → C 递进

## 改写注意事项

### 避免的问题

1. **洗稿嫌疑**
   - 仅替换个别词汇
   - 仅调换段落顺序
   - 保留原文特殊表达

2. **信息失真**
   - 改变原文核心观点
   - 遗漏关键信息
   - 添加不实内容

3. **风格不一致**
   - 前后语气不统一
   - 专业度忽高忽低
   - 人称混乱

### 改写质量标准

| 维度 | 标准 | 检查方法 |
|------|------|----------|
| 原创度 | >70% | 查重工具检测 |
| 信息完整 | 100% | 对照原文核查 |
| 表达流畅 | 无语病 | 通读检查 |
| 风格统一 | 一致 | 整体审视 |
