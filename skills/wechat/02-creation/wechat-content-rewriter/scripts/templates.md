# 内容改写模板

## 改写提示词模板

```
请将以下内容改写，要求：
1. 保持原文核心观点不变
2. 改变表达方式和句式结构
3. 调整段落顺序和逻辑组织
4. 替换同义词和表达方式
5. 适配{目标平台}的风格

【原文】
{粘贴原文}

【改写要求】
- 风格：{专业/轻松/幽默/温暖}
- 字数：{增加/减少/保持}约{X}字
- 重点强调：{需要突出的内容}
```

## 不同改写场景模板

### 场景1：跨平台改写

```
将以下{原平台}内容改写为适合{目标平台}的风格：

【原文】
{内容}

【改写要求】
- 调整语气：{正式→口语/口语→正式}
- 调整长度：{长文→短文/短文→长文}
- 调整结构：{适配目标平台阅读习惯}
```

### 场景2：风格转换

```
将以下内容改写为{目标风格}：

【原文】
{内容}

【目标风格】
- 专业严谨：使用专业术语，逻辑严密
- 轻松幽默：加入比喻、调侃，降低阅读门槛
- 温暖治愈：情感化表达，引发共鸣
- 犀利深刻：观点鲜明，直击要害
```

### 场景3：精简压缩

```
将以下内容精简为{X}字以内，保留核心信息：

【原文】
{内容}

【保留要点】
1. {必须保留的信息1}
2. {必须保留的信息2}
3. {必须保留的信息3}
```

### 场景4：扩写丰富

```
将以下内容扩写为{X}字左右，增加细节和案例：

【原文】
{内容}

【扩写方向】
- 增加案例：{相关案例}
- 增加数据：{支撑数据}
- 增加细节：{具体描述}
```

## 改写检查清单

- [ ] 核心观点是否保持一致
- [ ] 表达方式是否有明显变化
- [ ] 句式结构是否重新组织
- [ ] 是否适配目标平台风格
- [ ] 是否有语法或表达错误
- [ ] 原创度是否足够高
