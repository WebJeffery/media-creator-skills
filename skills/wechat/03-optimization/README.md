# 公众号 - 优化阶段 Skills

优化阶段决定内容的点击率和传播效果。

## 本阶段目标

- 📝 生成爆款标题，提升打开率
- 🎨 优化排版布局，提升阅读体验
- ✅ 检查内容规范，避免违规风险
- 📊 准备发布素材，确保发布顺利

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [title-generator](./title-generator/) | 爆款标题生成，基于10w+爆文规律 | ✅ 可用 |
| [layout-optimizer](./layout-optimizer/) | 排版优化，提升移动端阅读体验 | ✅ 可用 |
| summary-writer | 摘要撰写，优化预览效果 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 title-generator 生成多个备选标题
2. 使用 layout-optimizer 优化文章排版
3. 撰写摘要（120字以内）
4. 准备封面图（单图900×383 或 三图200×200）
5. 最终检查，确认发布
```

## 公众号优化要点

### 标题优化公式

| 公式 | 示例 | 适用场景 |
|------|------|----------|
| 数字+结果 | 「3个方法让你效率翻倍」 | 干货教程 |
| 痛点+方案 | 「总是拖延？试试这个方法」 | 解决问题 |
| 悬念+好奇 | 「90%的人都不知道的真相」 | 观点输出 |
| 人群+利益 | 「职场新人必看的5条建议」 | 精准定位 |
| 对比+反转 | 「从月薪3k到3w，我只做了这件事」 | 故事案例 |

### 排版规范

| 项目 | 建议 |
|------|------|
| 正文字号 | 15-16px |
| 行间距 | 1.75-2倍 |
| 段间距 | 空一行 |
| 字间距 | 1-2px |
| 两端缩进 | 16-32px |
| 配色 | 主色不超过3种 |

### 发布前检查清单

- [ ] 标题是否吸引人（64字符以内）
- [ ] 摘要是否精炼（120字符以内）
- [ ] 封面图是否清晰美观
- [ ] 正文是否有错别字
- [ ] 排版是否适合移动端阅读
- [ ] 是否有敏感词或违规内容
- [ ] 原创声明是否勾选
- [ ] 发布时间是否最佳（8:00/12:00/20:00）
