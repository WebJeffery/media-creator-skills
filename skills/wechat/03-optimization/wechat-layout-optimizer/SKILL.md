# 公众号排版优化助手

优化公众号文章排版，提升移动端阅读体验，输出适合直接粘贴到公众号编辑器的格式。

## 适用场景

- 优化已有文章的排版结构
- 为文章添加合适的格式标记（加粗、引用、分隔线等）
- 生成适合公众号的排版方案建议

## 使用方法

```
优化以下公众号文章的排版：
[粘贴文章内容]
```

```
给这篇文章生成排版建议，风格偏商务专业
```

## 排版规范

### 字体与字号
- 正文：15-16px，行间距 1.75-2 倍
- 标题：18-20px，加粗
- 注释/来源：12-13px，灰色

### 段落格式
- 段落间空一行
- 每段不超过 4-5 行（移动端）
- 重要内容加粗或使用引用框

### 视觉元素
- 小标题前使用 emoji 或编号标记
- 适当使用分隔线区分章节
- 关键数据可使用引用块突出

### 色彩建议
- 正文：#3f3f3f（深灰，护眼）
- 强调：#1e6fff 或品牌色
- 引用/注释：#888888

## 输出格式

1. **排版后的文章**（含格式标记）
2. **排版说明**（使用了哪些排版技巧）
3. **配图建议**（建议在哪些位置插入图片）

## 最佳实践

- **移动端优先**：90% 以上读者在手机阅读，确保小屏体验
- **呼吸感**：留白比内容更重要，不要堆砌文字
- **一致性**：全文保持统一的排版风格
- **引导视线**：用加粗、颜色、引用框引导读者注意力
