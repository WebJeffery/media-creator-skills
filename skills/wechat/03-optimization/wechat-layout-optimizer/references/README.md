# 公众号排版参考资料

## 排版原则

### 移动端阅读特点

1. **屏幕小**：内容要精简，避免大段文字
2. **碎片化**：用户注意力有限，重点要突出
3. **滑动阅读**：小标题分段，便于快速浏览
4. **环境多变**：字号要适中，对比度要够

### 排版三原则

| 原则 | 说明 | 实践 |
|------|------|------|
| 对齐 | 元素对齐，视觉整齐 | 统一左对齐或居中 |
| 对比 | 层次分明，重点突出 | 标题加粗、颜色区分 |
| 留白 | 呼吸感，不拥挤 | 段落间空行、边距 |

## 图片规范

### 封面图尺寸

| 类型 | 尺寸 | 比例 |
|------|------|------|
| 单图封面 | 900×383 | 2.35:1 |
| 三图封面 | 200×200 | 1:1 |
| 小图封面 | 200×200 | 1:1 |

### 正文图片

| 场景 | 建议宽度 | 说明 |
|------|----------|------|
| 全宽图 | 900px | 横向铺满 |
| 居中图 | 600-800px | 两侧留白 |
| 配图 | 400-600px | 配合文字 |

### 图片优化

- 格式：JPG（照片）、PNG（图标/截图）
- 大小：单张不超过2MB
- 清晰度：避免模糊、拉伸
- 水印：去除其他平台水印

## 常见排版问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 文字太密 | 行间距不够 | 增加到1.75-2倍 |
| 阅读疲劳 | 段落太长 | 每段3-5行，加小标题 |
| 重点不突出 | 缺少强调 | 加粗、变色、加框 |
| 视觉混乱 | 颜色太多 | 控制在3种以内 |
| 图片变形 | 比例不对 | 保持原始比例 |

## 优秀排版案例特征

1. **层次清晰**：标题、正文、引用层次分明
2. **重点突出**：核心观点加粗或变色
3. **适度留白**：段落间有呼吸感
4. **配色统一**：整体色调和谐
5. **图文搭配**：图片与文字相得益彰
