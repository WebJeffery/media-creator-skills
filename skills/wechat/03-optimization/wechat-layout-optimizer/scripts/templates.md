# 公众号排版模板

## 排版样式模板

### 基础排版参数

```css
/* 推荐排版参数 */
正文字号：15-16px
行间距：1.75-2倍
字间距：1-2px
段间距：空一行
两端缩进：16-32px
页边距：16px
```

### 标题样式

```markdown
# 一级标题
字号：20-24px | 加粗 | 居中或左对齐

## 二级标题
字号：17-18px | 加粗 | 左对齐

### 三级标题
字号：16px | 加粗 | 左对齐
```

### 正文样式

```markdown
【普通段落】
字号：15-16px
颜色：#3f3f3f 或 #595959
行高：2倍

【重点内容】
加粗 + 颜色强调（如 #007AFF）

【引用内容】
使用引用框样式
背景色：#f7f7f7
左边框：3px 主题色
```

## 常用排版元素

### 分隔线样式

```
───────────────────
• • •
─ ─ ─ ─ ─ ─ ─ ─ ─
◆ ◆ ◆
```

### 序号样式

```
数字序号：1. 2. 3.
圆点序号：• • •
emoji序号：1️⃣ 2️⃣ 3️⃣
符号序号：▶ ▷ ►
```

### 强调样式

```
【加粗】重点内容
「引号」特殊词汇
『双引号』书名/标题
→ 箭头指向
★ 星标重点
```

## 排版检查清单

- [ ] 字号是否适合移动端阅读（15-16px）
- [ ] 行间距是否足够（1.75-2倍）
- [ ] 段落间是否有空行
- [ ] 重点内容是否有加粗标注
- [ ] 配色是否统一（不超过3种主色）
- [ ] 图片是否清晰、大小合适
- [ ] 整体是否有呼吸感（留白）
