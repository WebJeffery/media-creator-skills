# 公众号爆款标题生成器

基于 10w+ 爆文标题规律，为公众号文章生成高点击率标题。

## 适用场景

- 为已写好的文章生成多个备选标题
- 根据主题/关键词生成标题
- 优化现有标题的点击率
- A/B 测试标题方案

## 使用方法

```
帮我生成5个公众号标题，主题是"职场沟通技巧"
```

```
优化这个标题："如何提升工作效率"，让它更有吸引力
```

## 爆款标题公式

### 1. 数字型
- 「{数字}个{方法/技巧}，让你{获得收益}」
- 示例：「7个早起习惯，让你一年多赚10万」

### 2. 痛点型
- 「为什么{痛点现象}？因为你{错误做法}」
- 示例：「为什么你总是存不下钱？因为你忽略了这3个细节」

### 3. 悬念型
- 「{某人/某事}{意外结果}，真相竟然是……」
- 示例：「月薪3千的她买了套房，真相竟然是……」

### 4. 对比型
- 「{A}和{B}的差距，就在于{关键因素}」
- 示例：「普通人和高手的差距，就在于这一个思维习惯」

### 5. 权威型
- 「{权威来源}：{核心观点}」
- 示例：「哈佛研究：成功人士都有这5个共同特征」

### 6. 时效型
- 「{时间节点}，{必须做的事}」
- 示例：「2025年下半年，这3个行业将迎来爆发」

## 输出格式

为每个标题提供：
1. 标题文本
2. 使用的公式类型
3. 预估吸引力评分（1-10）
4. 优化建议

## 平台规则

- 标题上限 64 个字符
- 避免标题党（与内容严重不符）
- 禁止使用"震惊"、"速看"等低质标题党词汇
- 不得包含虚假信息或夸大宣传

## 最佳实践

- **控制字数**：22-35 字为最佳长度，手机端显示完整
- **前置关键词**：核心信息放在标题前半部分
- **制造信息差**：让读者感到"不点开就亏了"
- **具体化**：用具体数字代替模糊描述
- **情绪共鸣**：触发好奇、焦虑、认同等情绪
