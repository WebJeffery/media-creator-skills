# 公众号标题参考资料

## 爆款标题研究

### 10w+ 标题特征分析

基于 500+ 篇 10w+ 阅读量文章标题分析：

| 特征 | 占比 | 说明 |
|------|------|------|
| 包含数字 | 67% | 具体数字比模糊描述更有吸引力 |
| 痛点切入 | 58% | 直击读者痛点引发共鸣 |
| 悬念设置 | 45% | 制造好奇心驱动点击 |
| 利益承诺 | 52% | 明确告知读者能获得什么 |
| 情感词汇 | 38% | 使用情感化词汇增强感染力 |

### 高点击率词汇

**数字类**
- 3、5、7、10（奇数更受欢迎）
- 90%、99%（高比例数字）
- 100%、翻倍（效果类数字）

**情感类**
- 震惊、意外、惊人
- 必看、必读、必学
- 终于、原来、竟然

**利益类**
- 免费、省钱、赚钱
- 快速、简单、轻松
- 秘密、真相、内幕

### 标题禁忌词

| 类型 | 禁忌词 | 原因 |
|------|--------|------|
| 绝对化 | 最好、第一、100% | 违反广告法 |
| 诱导性 | 不转不是中国人 | 平台限制 |
| 低俗类 | 色情暗示词汇 | 违规风险 |
| 虚假类 | 夸大效果词汇 | 标题党嫌疑 |

## 标题A/B测试

### 测试方法

1. **准备2-3个备选标题**
2. **小范围测试**：先发给部分用户
3. **数据对比**：比较打开率
4. **选择最优**：使用数据最好的标题

### 测试维度

| 维度 | 变量 | 示例 |
|------|------|------|
| 数字 | 有无数字 | "方法" vs "3个方法" |
| 人群 | 有无定位 | "技巧" vs "职场人必看的技巧" |
| 情感 | 情感强度 | "重要" vs "太重要了" |
| 长度 | 长短标题 | 简洁版 vs 详细版 |

## 各类型文章标题参考

### 干货文标题

- {数字}个{技巧}，{效果}
- {人群}必看的{主题}指南
- 如何{目标}？这篇讲透了

### 观点文标题

- 关于{话题}，我有{数字}点想说
- {现象}背后的真相
- 为什么我{观点}

### 故事文标题

- 从{起点}到{终点}，我经历了什么
- {人物}的{经历}，看哭了{数字}万人
- 那个{描述}的人，后来怎么样了
