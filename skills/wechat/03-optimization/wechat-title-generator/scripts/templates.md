# 公众号爆款标题模板

## 标题公式库

### 1. 数字型标题

```
{数字}个{方法/技巧}，让你{获得结果}
{数字}种{类型}，{效果描述}
{数字}分钟学会{技能}
从{数字A}到{数字B}，我只用了{方法}
```

**示例**：
- 3个方法，让你的文章阅读量翻倍
- 5种思维方式，决定你的人生高度
- 10分钟学会高效阅读
- 从月薪3000到30000，我只用了这个方法

### 2. 痛点型标题

```
为什么你{痛点}？原因在这里
{痛点}？试试这{数字}个方法
90%的人都不知道的{领域}真相
别再{错误做法}了，{正确做法}才是关键
```

**示例**：
- 为什么你总是存不下钱？原因在这里
- 总是拖延？试试这3个方法
- 90%的人都不知道的护肤真相
- 别再熬夜了，早起才是成功的关键

### 3. 悬念型标题

```
{结果}，背后的原因让人意外
{现象}，真相竟然是...
看完这篇，你会重新认识{主题}
{人物/事件}的{秘密}，终于公开了
```

**示例**：
- 他年入百万，背后的原因让人意外
- 这个行业正在消失，真相竟然是...
- 看完这篇，你会重新认识时间管理
- 顶级销售的成交秘密，终于公开了

### 4. 对比型标题

```
{A}和{B}的区别，看完你就懂了
从{负面状态}到{正面状态}，我做对了{事情}
{普通人}vs{成功人}：{差异点}
同样是{身份}，为什么{差距}
```

**示例**：
- 穷人思维和富人思维的区别，看完你就懂了
- 从社恐到社牛，我做对了这3件事
- 普通员工vs优秀员工：差距在这5点
- 同样是写作，为什么他能月入过万

### 5. 权威型标题

```
{权威来源}都在用的{方法/产品}
{专家/名人}推荐的{数字}个{类型}
{行业}内部人才知道的{秘密}
{大厂/名校}都在学的{技能}
```

**示例**：
- 哈佛教授都在用的学习方法
- 巴菲特推荐的5本投资书籍
- 金融行业内部人才知道的理财秘密
- 谷歌都在学的高效工作法

## 标题优化检查清单

- [ ] 字数是否在64字符以内
- [ ] 是否包含核心关键词
- [ ] 是否有吸引点击的元素
- [ ] 是否避免了标题党嫌疑
- [ ] 是否与内容相符
