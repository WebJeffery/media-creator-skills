# 公众号 - 分析阶段 Skills

分析阶段帮助创作者了解内容表现，持续优化创作策略。

## 本阶段目标

- 📊 追踪内容数据，了解文章表现
- 📈 分析用户行为，优化内容策略
- 🔎 监控竞品动态，学习优秀案例
- 🔮 预测内容趋势，提前布局选题

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| performance-tracker | 文章数据追踪，监控阅读/分享/留言 | 🔲 规划中 |
| engagement-analyzer | 互动数据分析，了解用户偏好 | 🔲 规划中 |
| competitor-monitor | 竞品账号监控，学习爆款规律 | 🔲 规划中 |

## 公众号核心数据指标

### 内容表现指标

| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 阅读量 | 文章被打开的次数 | 粉丝数×5%+ |
| 打开率 | 阅读量/送达人数 | 3%+ |
| 分享率 | 分享次数/阅读量 | 2%+ |
| 在看率 | 在看数/阅读量 | 1%+ |
| 留言率 | 留言数/阅读量 | 0.5%+ |
| 收藏率 | 收藏数/阅读量 | 1%+ |

### 数据分析维度

```
┌─────────────────────────────────────────────────┐
│                  内容分析框架                    │
├─────────────────────────────────────────────────┤
│  时间维度：发布时间 vs 阅读高峰                  │
│  内容维度：主题类型 vs 数据表现                  │
│  标题维度：标题风格 vs 打开率                    │
│  用户维度：新老用户 vs 阅读来源                  │
│  传播维度：一次传播 vs 二次传播                  │
└─────────────────────────────────────────────────┘
```

### 优化迭代建议

1. **高阅读低分享**：内容有吸引力但缺乏传播动机，增加社交货币
2. **低阅读高分享**：标题不够吸引，但内容质量高，优化标题
3. **高留言低在看**：引发讨论但不够认同，调整观点表达
4. **阅读量下降**：检查发布时间、标题质量、内容匹配度
