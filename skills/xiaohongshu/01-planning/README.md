# 小红书 - 策划阶段 Skills

策划阶段决定笔记的选题方向和内容定位。

## 本阶段目标

- 🔥 发现热门话题，抓住流量机会
- 👥 分析目标用户，明确内容定位
- 📋 规划笔记选题，建立内容矩阵
- 🏷️ 研究热门标签，提升曝光潜力

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| topic-planner | 小红书选题规划，发现热门方向 | 🔲 规划中 |
| niche-finder | 细分赛道分析，找到蓝海领域 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 common/hot-topic-tracker 追踪小红书热点
2. 使用 common/audience-analyzer 分析目标用户
3. 研究同类爆款笔记，提炼选题方向
4. 使用 common/content-calendar 规划发布节奏
```

## 小红书选题要点

### 热门内容赛道

| 赛道 | 细分方向 | 内容形式 |
|------|----------|----------|
| 美妆护肤 | 测评、教程、好物分享 | 图文/视频 |
| 穿搭时尚 | OOTD、搭配技巧、购物分享 | 图文为主 |
| 美食探店 | 探店、食谱、零食测评 | 图文/视频 |
| 家居生活 | 收纳、改造、好物推荐 | 图文为主 |
| 旅行出游 | 攻略、打卡、酒店测评 | 图文/视频 |
| 母婴育儿 | 好物、经验、成长记录 | 图文/视频 |
| 健身减脂 | 教程、饮食、打卡记录 | 视频为主 |

### 选题灵感来源

1. **小红书搜索框**：输入关键词看联想词
2. **发现页热门**：观察平台推荐内容
3. **同类博主**：分析爆款笔记选题
4. **评论区**：用户问题就是选题
5. **生活观察**：真实体验和发现

### 爆款选题公式

| 公式 | 示例 |
|------|------|
| 人群+痛点 | 学生党平价护肤 |
| 场景+解决方案 | 通勤妆容5分钟搞定 |
| 数字+品类 | 5款回购无限的面霜 |
| 对比+结果 | 从黄黑皮到冷白皮 |
| 时间+效果 | 坚持30天的变化 |
