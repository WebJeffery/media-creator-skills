# 小红书 - 创作阶段 Skills

创作阶段是产出高质量笔记内容的核心环节。

## 本阶段目标

- ✍️ 撰写种草笔记，输出有价值的内容
- 💡 设计吸睛开头，提升笔记点击率
- 📝 优化文案表达，适配平台风格
- 😊 添加 emoji 表情，增强阅读体验

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [note-writer](./note-writer/) | 小红书笔记写作，自动适配平台风格 | ✅ 可用 |
| [hook-writer](./hook-writer/) | 开头金句生成，提升笔记点击率 | ✅ 可用 |
| emoji-enhancer | Emoji 优化，提升视觉吸引力 | 🔲 规划中 |

## 推荐工作流

```
1. 使用 common/copywriting-formulas 选择文案框架
2. 使用 hook-writer 生成吸睛开头
3. 使用 note-writer 撰写笔记正文
4. 添加 emoji 和表情，优化阅读体验
```

## 小红书写作要点

### 笔记结构

```
┌─────────────────────────────────────┐
│  开头（前2行）- 决定是否继续阅读      │
│  • 痛点共鸣 / 结果展示 / 悬念设置    │
├─────────────────────────────────────┤
│  正文（分点展开）- 价值输出          │
│  • 用 emoji 分隔段落                │
│  • 每段 2-3 行，保持简洁            │
│  • 口语化表达，像朋友聊天            │
├─────────────────────────────────────┤
│  结尾（互动引导）- 促进互动          │
│  • 提问引导评论                     │
│  • 收藏/关注引导                    │
└─────────────────────────────────────┘
```

### 不同类型笔记写法

| 类型 | 核心要素 | 字数建议 |
|------|----------|----------|
| 好物分享 | 真实体验、使用感受、适合人群 | 300-600 |
| 教程攻略 | 步骤清晰、图文对应、可操作 | 400-800 |
| 测评对比 | 客观公正、维度清晰、有结论 | 500-800 |
| 经验分享 | 真实故事、干货总结、情感共鸣 | 400-700 |
| 探店打卡 | 地址信息、环境描述、推荐菜品 | 300-500 |

### 小红书文案风格

**DO ✅**
- 口语化、像朋友聊天
- 用 emoji 增加活力
- 真实体验、有细节
- 分段清晰、易阅读

**DON'T ❌**
- 过于正式、像说明书
- 纯文字、没有表情
- 虚假夸大、无依据
- 大段文字、难阅读
