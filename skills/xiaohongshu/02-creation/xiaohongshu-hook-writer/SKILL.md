# 小红书开头金句生成器

为小红书笔记生成吸引人的开头文案，提升用户展开阅读的概率。

## 适用场景

- 为笔记生成多个开头方案
- 优化已有笔记的开头
- 学习不同类型的开头写法

## 使用方法

```
帮我写5个小红书笔记开头，主题是"租房改造"
```

```
优化这个开头："今天分享一下我的护肤心得"，让它更吸引人
```

## 开头公式

### 1. 痛点共鸣型
> 是不是每次{痛点场景}都{痛苦感受}？我之前也是！直到发现了{解决方案}...

### 2. 结果前置型
> {惊人结果}！！！就靠{方法}，{时间}就{达成效果}

### 3. 反常识型
> {常见认知}？大错特错！{正确做法}才是关键

### 4. 身份代入型
> {身份标签}姐妹看过来！这篇{内容类型}你一定需要

### 5. 数据冲击型
> 花了{金额/时间}，试了{数量}款{产品}，终于找到了{最佳选择}

### 6. 故事悬念型
> 上周{发生了什么事}，没想到{意外结果}...（往下看你就知道了）

### 7. 直接种草型
> 天花板级别的{品类}！用过的都说好，不允许还有人不知道！

## 输出格式

每个开头包含：
1. 开头文案（2-3 行）
2. 使用的公式类型
3. 适合的笔记类型
4. 吸引力评分（1-10）

## 平台规则

- 开头前 2 行在信息流中可见，是决定用户是否点击的关键
- 避免虚假承诺和夸大效果
- 不使用极限词和绝对化表述

## 最佳实践

- **前 2 行定生死**：信息流只展示前 2 行，必须在这里抓住注意力
- **制造信息差**：让用户觉得"不看完就亏了"
- **情绪先行**：先触发情绪（好奇、共鸣、惊讶），再给信息
- **口语化**：像和朋友聊天，不要像写作文
