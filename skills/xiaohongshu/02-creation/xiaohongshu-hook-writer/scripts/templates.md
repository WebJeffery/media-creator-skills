# 小红书开头金句模板

## 开头公式库

### 1. 痛点共鸣型

```
姐妹们！是不是也有这个困扰...
之前我也是{痛点}，直到...
如果你也{痛点}，一定要看完！
{痛点}的姐妹看过来！
```

**示例**：
- 姐妹们！是不是也觉得护肤品越用越多，皮肤却没变好？
- 之前我也是怎么减都减不下来，直到发现了这个方法
- 如果你也经常熬夜爆痘，一定要看完！

### 2. 效果展示型

```
用了{时间}，效果真的绝了！
从{之前}到{之后}，我只做了这{数字}步
这个{产品/方法}真的太香了！
{效果}的秘密，今天全部告诉你
```

**示例**：
- 用了一个月，皮肤真的白了一个度！
- 从黄黑皮到冷白皮，我只做了这3步
- 这个面霜真的太香了！回购第5瓶

### 3. 好奇悬念型

```
你们绝对想不到...
原来{认知}是错的！
{数字}%的人都不知道的{秘密}
千万别{错误做法}！
```

**示例**：
- 你们绝对想不到这个平价货这么好用
- 原来我们一直都洗错脸了！
- 90%的人都不知道的防晒真相
- 千万别这样敷面膜！

### 4. 身份认同型

```
作为一个{身份}，必须分享这个
{身份}姐妹看过来！
终于找到适合{身份}的{产品}了
{身份}的{场景}神器
```

**示例**：
- 作为一个油皮，必须分享这个控油神器
- 学生党姐妹看过来！平价好物合集
- 终于找到适合敏感肌的面霜了
- 打工人的早起神器，5分钟出门

### 5. 数字吸引型

```
{数字}款{品类}，亲测有效！
花了{金额}，终于找到了...
{数字}年{经验}，总结出这{数字}点
坚持{时间}，变化惊人
```

**示例**：
- 5款平价面霜，亲测有效！
- 花了3000块，终于找到适合我的护肤品
- 护肤10年，总结出这5点经验
- 坚持早起30天，变化惊人

## 开头写作技巧

1. **前两行定生死**：开头要直接、有冲击力
2. **制造共鸣**：让读者觉得"说的就是我"
3. **承诺价值**：告诉读者看完能得到什么
4. **口语化**：像朋友聊天一样自然
5. **加emoji**：增加视觉吸引力
