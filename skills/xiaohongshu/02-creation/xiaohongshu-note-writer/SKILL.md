# 小红书笔记写作助手

帮助创作者撰写符合小红书平台风格的笔记，自动适配平台调性，提升互动率和曝光量。

## 适用场景

- 撰写种草/测评/教程类笔记
- 将产品信息转化为小红书风格文案
- 生成不同类型的笔记（图文笔记、视频文案）
- 优化已有笔记的文案表达

## 使用方法

```
帮我写一篇小红书笔记，种草戴森吹风机
```

```
写一篇小红书教程笔记，主题是"5分钟快手早餐"，风格活泼可爱
```

## 笔记类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 种草笔记 | 真实体验感、生活化表达 | 产品推荐、好物分享 |
| 教程笔记 | 步骤清晰、实操性强 | 美妆教程、做饭教程、DIY |
| 测评笔记 | 客观对比、数据支撑 | 产品对比、横向测评 |
| 攻略笔记 | 信息密度高、收藏价值 | 旅行攻略、省钱攻略 |
| 日常分享 | 真实感、共鸣感 | 生活记录、心得感悟 |
| OOTD | 穿搭展示、风格标签 | 穿搭分享、风格推荐 |

## 小红书文案风格要素

1. **emoji 点缀**：每段开头或关键词旁加 emoji，增加视觉活力
2. **口语化表达**：像和闺蜜聊天一样自然，避免书面语
3. **感叹句/疑问句**：增加情绪感染力
4. **分行排版**：短句分行，一行一个信息点
5. **关键词加粗**：核心卖点或关键信息用【】或加粗标注

## 输出格式

1. **标题**（含 emoji，20 字以内）
2. **正文**（小红书风格排版，500-1000 字）
3. **话题标签**（5-10 个相关话题）
4. **封面文字建议**（适合做封面的核心文案）

## 平台规则

- 笔记正文上限 1000 字
- 标题上限 20 个字符
- 最多 30 个话题标签
- 图片最多 18 张（建议 6-9 张）
- 禁止出现微信号、二维码等导流信息
- 避免绝对化用语和虚假宣传
- 不得使用"最好"、"第一"、"100% 有效"等极限词

## 最佳实践

- **第一句话是关键**：前 2 行决定用户是否展开阅读
- **真实感 > 完美感**：小红书用户偏好真实体验分享
- **提供价值**：让读者觉得"收藏了以后用得上"
- **互动引导**：结尾设置问题或投票，提升评论量
- **发布时间**：工作日 12:00-14:00、18:00-22:00 为黄金时段
