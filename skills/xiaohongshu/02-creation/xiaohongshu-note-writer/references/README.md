# 小红书笔记参考资料

本目录收集小红书写作的参考资料和最佳实践。

## 爆款笔记特征研究

### 高互动笔记共性

基于对 500+ 篇万赞笔记的分析：

| 特征 | 占比 | 说明 |
|------|------|------|
| emoji 开头 | 82% | 标题或正文以 emoji 开头 |
| 数字标题 | 65% | 标题中包含具体数字 |
| 口语化表达 | 91% | 像朋友聊天的语气 |
| 分段清晰 | 87% | 使用 emoji 或符号分段 |
| 互动引导 | 73% | 结尾有提问或引导 |

### 不同品类爆款特征

| 品类 | 核心要素 | 图片要求 |
|------|----------|----------|
| 美妆护肤 | 真实效果、使用感受 | 产品图+效果对比 |
| 穿搭时尚 | 搭配技巧、身材参考 | 全身穿搭图 |
| 美食探店 | 地址信息、推荐菜品 | 食物特写+环境 |
| 家居生活 | 实用性、改造前后 | 整体效果+细节 |
| 旅行攻略 | 实用信息、省钱技巧 | 美景+实用信息图 |

## 小红书平台规则

### 内容规范

- 标题上限：20 字符
- 正文上限：1000 字
- 话题标签：最多 30 个
- 图片数量：最多 18 张（建议 6-9 张）
- 视频时长：最长 15 分钟

### 违规红线

- 虚假宣传、夸大效果
- 搬运抄袭他人内容
- 引流到其他平台
- 低俗色情内容
- 医疗健康类违规宣传

### 最佳发布时间

| 时段 | 适合内容 | 原因 |
|------|----------|------|
| 7:00-9:00 | 早安、日常 | 起床刷手机 |
| 12:00-14:00 | 美食、穿搭 | 午休时间 |
| 18:00-20:00 | 护肤、好物 | 下班放松 |
| 21:00-23:00 | 情感、干货 | 睡前深度阅读 |
