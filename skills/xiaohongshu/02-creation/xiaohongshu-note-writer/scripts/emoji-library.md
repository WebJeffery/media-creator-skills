## 使用技巧

1. **开头放 emoji**：在标题和正文开头放 emoji，更醒目
2. **分段用 emoji**：用 emoji 作为段落分隔符
3. **不要过多**：每段 1-2 个 emoji 即可，过多显得杂乱
4. **风格统一**：同一篇笔记使用风格一致的 emoji
5. **适配内容**：根据笔记主题选择相关 emoji
