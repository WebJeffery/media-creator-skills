# 小红书 - 优化阶段 Skills

优化阶段决定笔记的曝光量和互动效果。

## 本阶段目标

- 📝 生成爆款标题，提升点击率
- 🏷️ 优化话题标签，增加曝光机会
- 🖼️ 设计封面图片，吸引用户停留
- ✅ 检查内容规范，避免限流风险

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| [title-generator](./title-generator/) | 爆款标题生成，含 emoji 和关键词优化 | ✅ 可用 |
| [tag-optimizer](./tag-optimizer/) | 话题标签优化，提升笔记曝光量 | ✅ 可用 |
| [cover-advisor](./cover-advisor/) | 封面设计建议，基于热门笔记分析 | ✅ 可用 |

## 推荐工作流

```
1. 使用 title-generator 生成多个备选标题
2. 使用 tag-optimizer 选择最佳话题标签
3. 使用 cover-advisor 获取封面设计建议
4. 最终检查，确认发布
```

## 小红书优化要点

### 标题优化

| 公式 | 示例 | 适用场景 |
|------|------|----------|
| emoji+数字+品类 | ✨5款平价面霜推荐 | 好物分享 |
| 人群+痛点 | 学生党必看！省钱护肤法 | 精准定位 |
| 效果+方法 | 从黄皮到白皮，我只做了这3步 | 经验分享 |
| 场景+解决 | 约会妆容｜5分钟出门 | 教程攻略 |
| 疑问+答案 | 为什么你的妆总是斑驳？ | ��识科普 |

### 标签策略

| 标签类型 | 数量建议 | 示例 |
|----------|----------|------|
| 核心标签 | 3-5个 | #护肤 #面霜推荐 |
| 人群标签 | 2-3个 | #学生党 #油皮 |
| 场景标签 | 2-3个 | #秋冬护肤 #换季 |
| 热门标签 | 1-2个 | #好物分享 #我的爱用物 |

### 封面设计要点

| 要素 | 建议 |
|------|------|
| 尺寸 | 3:4 竖图（1080×1440） |
| 文字 | 大字突出、不超过10字 |
| 配色 | 明亮清新、对比鲜明 |
| 人物 | 真人出镜提升信任感 |
| 产品 | 清晰展示、排列整齐 |

### 发布前检查清单

- [ ] 标题是否吸引人（20字以内）
- [ ] 封面是否清晰美观
- [ ] 正文是否有错别字
- [ ] emoji 使用是否恰当
- [ ] 话题标签是否完整（建议10-15个）
- [ ] 是否有敏感词或违规内容
- [ ] 图片是否有水印需要去除
- [ ] 发布时间是否最佳（12:00-14:00/18:00-22:00）
