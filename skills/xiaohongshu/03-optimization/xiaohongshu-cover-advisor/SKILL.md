# 小红书封面设计建议

基于热门笔记的视觉分析，为小红书笔记提供封面设计建议，提升点击率。

## 适用场景

- 为笔记设计封面方案
- 优化现有封面的视觉效果
- 了解不同品类的封面设计趋势

## 使用方法

```
我要发一篇关于"家居改造"的笔记，给我封面设计建议
```

```
分析一下美妆类笔记的热门封面风格
```

## 封面设计要素

### 尺寸规范
- **推荐比例**：3:4（竖版，信息流最大展示面积）
- **推荐尺寸**：1080×1440px
- **备选比例**：1:1（正方形）、4:3（横版）

### 文字排版
- **标题文字**：大字醒目，6-12 个字为佳
- **字体选择**：无衬线体为主，手写体增加亲和力
- **文字位置**：居中或上 1/3 处，避免被头像遮挡
- **对比度**：文字与背景需有足够对比度

### 色彩搭配
| 品类 | 推荐色系 | 说明 |
|------|----------|------|
| 美妆护肤 | 粉色系、奶油色 | 温柔、精致感 |
| 穿搭时尚 | 黑白灰、莫兰迪 | 高级、质感 |
| 美食 | 暖色系、明亮色 | 食欲感、温馨 |
| 家居 | 木色、白色、绿色 | 自然、舒适 |
| 旅行 | 蓝色、绿色、明亮色 | 清新、向往感 |
| 学习干货 | 蓝色、白色 | 专业、信赖感 |

### 封面类型
1. **纯图片型**：高质量实拍图，适合美食/旅行/穿搭
2. **图文结合型**：图片 + 大字标题，适合教程/攻略
3. **拼图对比型**：Before/After 对比，适合改造/测评
4. **纯文字型**：纯色背景 + 文字，适合干货/观点

## 输出格式

1. **封面方案**（含布局描述、色彩建议、文字内容）
2. **设计要点**（为什么这样设计）
3. **避坑提醒**（常见设计错误）

## 最佳实践

- **3 秒法则**：用户在信息流中只看 3 秒，封面要一眼抓住注意力
- **信息聚焦**：封面只传达 1 个核心信息，不要堆砌
- **风格统一**：同一账号保持统一的封面风格，建立视觉识别
- **测试迭代**：同一内容尝试不同封面，观察数据差异
