# 小红书封面设计模板

## 封面类型模板

### 1. 产品展示型

```
【布局】
┌─────────────────┐
│    大标题文字    │
│   (2-3个关键词)  │
├─────────────────┤
│                 │
│   产品排列图     │
│   (整齐摆放)     │
│                 │
└─────────────────┘

【要点】
- 产品摆放整齐，有序排列
- 背景干净，纯色或简单纹理
- 文字突出，字号大
- 光线充足，产品清晰
```

### 2. 效果对比型

```
【布局】
┌────────┬────────┐
│ Before │ After  │
│  之前  │  之后  │
├────────┴────────┤
│    效果描述     │
└─────────────────┘

【要点】
- 对比明显，差异清晰
- 拍摄角度一致
- 光线条件相似
- 文字标注清楚
```

### 3. 教程步骤型

```
【布局】
┌─────────────────┐
│    教程标题     │
│  (数字+效果)    │
├────┬────┬────┤
│ 1  │ 2  │ 3  │
│步骤│步骤│步骤│
└────┴────┴────┘

【要点】
- 步骤清晰，数字标注
- 每步图片简洁
- 整体风格统一
- 文字说明简短
```

### 4. 人物出镜型

```
【布局】
┌─────────────────┐
│                 │
│    人物照片     │
│   (半身/全身)   │
│                 │
├─────────────────┤
│   标题文字      │
└─────────────────┘

【要点】
- 表情自然，有感染力
- 光线柔和，肤色好看
- 背景简洁不杂乱
- 文字不遮挡人物
```

## 封面尺寸规范

| 类型 | 尺寸 | 比例 |
|------|------|------|
| 竖图（推荐） | 1080×1440 | 3:4 |
| 方图 | 1080×1080 | 1:1 |
| 横图 | 1440×1080 | 4:3 |

## 封面文字规范

| 元素 | 建议 |
|------|------|
| 字数 | 不超过10个字 |
| 字号 | 大而醒目 |
| 字体 | 清晰易读 |
| 颜色 | 与背景对比强 |
| 位置 | 上方或下方 |

## 配色方案

### 清新风格
- 主色：#E8F5E9（浅绿）
- 文字：#2E7D32（深绿）

### 温暖风格
- 主色：#FFF3E0（浅橙）
- 文字：#E65100（深橙）

### 高级风格
- 主色：#F5F5F5（浅灰）
- 文字：#212121（深灰）

### 少女风格
- 主色：#FCE4EC（浅粉）
- 文字：#C2185B（深粉）
