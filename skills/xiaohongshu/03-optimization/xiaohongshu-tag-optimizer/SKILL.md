# 小红书话题标签优化

为小红书笔记推荐最优话题标签组合，提升笔记在搜索和推荐中的曝光量。

## 适用场景

- 为新笔记选择合适的话题标签
- 优化已有笔记的标签组合
- 分析特定品类的热门标签

## 使用方法

```
我写了一篇关于"秋冬穿搭"的笔记，帮我推荐话题标签
```

```
优化这些标签：#穿搭 #秋冬 #时尚，让曝光更高
```

## 标签策略

### 三层标签金字塔

| 层级 | 类型 | 数量 | 说明 |
|------|------|------|------|
| 顶层 | 大流量标签 | 2-3 个 | 百万级话题，如 #穿搭分享 #美食推荐 |
| 中层 | 精准标签 | 3-4 个 | 十万级话题，如 #秋冬穿搭 #韩系穿搭 |
| 底层 | 长尾标签 | 2-3 个 | 万级话题，竞争小易上热门 |

### 标签类型
- **品类标签**：#护肤 #美妆 #穿搭
- **场景标签**：#通勤穿搭 #约会妆容 #宿舍好物
- **人群标签**：#学生党 #上班族 #新手妈妈
- **风格标签**：#极简风 #韩系 #法式
- **热点标签**：当前平台热门话题

## 输出格式

```
📌 推荐标签组合（共 8 个）

🔥 大流量标签：
#穿搭分享 (1200w 浏览) | #每日穿搭 (800w 浏览)

🎯 精准标签：
#秋冬穿搭 (350w 浏览) | #韩系穿搭 (200w 浏览) | #显瘦穿搭 (180w 浏览)

💎 长尾标签：
#小个子秋冬穿搭 (15w 浏览) | #通勤穿搭灵感 (8w 浏览) | #梨形身材穿搭 (12w 浏览)
```

## 平台规则

- 每篇笔记最多 30 个话题标签
- 标签需与内容相关，不相关标签会被降权
- 避免使用被限流的标签

## 最佳实践

- **数量适中**：8-15 个标签为最佳，过多反而降权
- **相关性优先**：标签必须与内容高度相关
- **大小搭配**：大流量标签获取曝光，长尾标签获取精准流量
- **定期更新**：关注平台热门标签变化，及时调整
