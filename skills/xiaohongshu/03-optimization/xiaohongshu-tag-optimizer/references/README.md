# 小红书话题标签参考资料

## 标签算法原理

### 标签的作用

1. **内容分类**：帮助平台理解笔记内容
2. **精准推荐**：匹配感兴趣的用户
3. **搜索曝光**：用户搜索时展示
4. **话题聚合**：进入话题页获得流量

### 标签权重因素

| 因素 | 权重 | 说明 |
|------|------|------|
| 相关性 | 高 | 标签与内容的匹配度 |
| 热度 | 中 | 标签的搜索和使用量 |
| 竞争度 | 中 | 该标签下的内容数量 |
| 位置 | 低 | 标签在列表中的位置 |

## 热门标签研究

### 2024年热门话题标签

| 品类 | 热门标签 | 浏览量级 |
|------|----------|----------|
| 美妆 | #好物分享 | 100亿+ |
| 护肤 | #护肤心得 | 50亿+ |
| 穿搭 | #今日穿搭 | 80亿+ |
| 美食 | #美食分享 | 60亿+ |
| 家居 | #家居好物 | 30亿+ |
| 旅行 | #旅行攻略 | 40亿+ |

### 长尾标签策略

**大标签**（竞争激烈，曝光大）
- #护肤 #穿搭 #美食

**中标签**（适中竞争，精准流量）
- #油皮护肤 #小个子穿搭 #探店

**小标签**（竞争小，精准用户）
- #油皮面霜推荐 #155小个子穿搭

### 标签组合公式

```
最佳组合 = 2-3个大标签 + 5-8个中标签 + 3-5个小标签

示例（护肤笔记）：
大标签：#护肤 #好物分享
中标签：#面霜推荐 #保湿面霜 #秋冬护肤 #油皮护肤 #平价护肤
小标签：#油皮面霜 #学生党面霜 #百元面霜
```

## 标签使用规范

### 平台规则

- 最多可添加 30 个标签
- 标签不计入正文字数
- 避免使用无关标签（可能被限流）
- 避免使用违规标签

### 标签禁忌

| 禁忌 | 后果 |
|------|------|
| 堆砌无关标签 | 降低推荐权重 |
| 使用违规标签 | 笔记被限流 |
| 标签与内容不符 | 用户体验差，影响互动 |
| 只用大热标签 | 竞争激烈，难以突围 |

## 标签效果追踪

### 数据指标

| 指标 | 说明 | 优化方向 |
|------|------|----------|
| 搜索曝光 | 通过搜索获得的曝光 | 优化关键词标签 |
| 推荐曝光 | 通过推荐获得的曝光 | 优化内容相关标签 |
| 话题页流量 | 从话题页进入的流量 | 参与热门话题 |

### 优化建议

1. 定期查看笔记数据，分析流量来源
2. 对比不同标签组合的效果
3. 关注平台热门话题变化
4. 学习同类爆款笔记的标签策略
