# 小红书话题标签模板

## 标签组合策略

### 标签分类

```
【核心标签】3-5个
直接描述内容主题的标签
示例：#护肤 #面霜推荐 #保湿

【人群标签】2-3个
定位目标用户群体
示例：#学生党 #油皮 #敏感肌

【场景标签】2-3个
使用场景或时间节点
示例：#秋冬护肤 #换季 #日常护肤

【热门标签】1-2个
平台热门话题
示例：#好物分享 #我的爱用物 #真实测评
```

### 标签数量建议

| 笔记类型 | 建议数量 | 说明 |
|----------|----------|------|
| 好物分享 | 10-15个 | 覆盖产品+人群+场景 |
| 教程攻略 | 8-12个 | 技能+人群+效果 |
| 探店打卡 | 10-15个 | 地点+品类+特色 |
| 穿搭分享 | 10-15个 | 风格+身材+场景 |

## 标签优化技巧

1. **关键词前置**：最重要的标签放前面
2. **大小搭配**：热门大标签+精准小标签
3. **避免重复**：不要用意思相近的标签
4. **定期更新**：关注平台热门标签变化
5. **测试优化**：对比不同标签组合的效果
