# 小红书爆款标题生成器

为小红书笔记生成高点击率标题，含 emoji 搭配和关键词优化，提升笔记曝光。

## 适用场景

- 为笔记生成多个备选标题
- 优化现有标题的点击率
- 针对特定品类生成标题模板

## 使用方法

```
生成5个小红书标题，主题是"平价护肤品推荐"
```

```
优化这个小红书标题："分享我的护肤步骤"
```

## 爆款标题公式

### 1. 数字 + 结果型
- 「{emoji}{数字}款{品类}，{效果描述}！」
- 示例：「✨5款平价面霜，用完皮肤嫩得能掐出水！」

### 2. 人群 + 痛点型
- 「{emoji}{人群}必看！{解决方案}」
- 示例：「💡学生党必看！月花200的护肤方案」

### 3. 对比反转型
- 「{emoji}从{负面状态}到{正面状态}，我只做了{方法}」
- 示例：「🔥从黄黑皮到冷白皮，我只做了这3步」

### 4. 权威背书型
- 「{emoji}{权威来源}都在用的{产品/方法}」
- 示例：「💄皮肤科医生都在用的5款防晒霜」

### 5. 紧迫感型
- 「{emoji}后悔没早知道的{数量}个{技巧}」
- 示例：「😭后悔没早知道的10个收纳神器」

### 6. 场景代入型
- 「{emoji}{场景}绝绝子！{核心亮点}」
- 示例：「🏠独居女生的房间改造，花500块效果绝了」

## 输出格式

每个标题包含：
1. 完整标题（含 emoji）
2. 标题公式类型
3. 核心关键词
4. 吸引力评分（1-10）

## 平台规则

- 标题上限 20 个字符
- emoji 不计入字数但不宜超过 3 个
- 避免纯大写或过多感叹号
- 禁止虚假宣传和极限词

## 最佳实践

- **emoji 前置**：标题开头放 emoji，在信息流中更醒目
- **关键词前置**：核心品类词放在前半部分，利于搜索
- **制造好奇**：让用户产生"点开看看"的冲动
- **具体数字**：用具体数字替代模糊描述
- **口语化**：像朋友推荐一样自然
