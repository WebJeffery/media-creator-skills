# 小红书标题参考资料

## 爆款标题研究

### 万赞笔记标题特征

| 特征 | 占比 | 说明 |
|------|------|------|
| emoji开头 | 82% | 在信息流中更醒目 |
| 包含数字 | 65% | 具体数字更有说服力 |
| 20字以内 | 95% | 简洁有力 |
| 关键词前置 | 78% | 核心词放前面 |
| 口语化 | 88% | 像朋友推荐 |

| 品类 | 热门关键词 |

| 护肤 | 平价、学生党、油皮、敏感肌、美白、抗老 |
| 彩妆 | 新手、日常、约会、通勤、持妆、显白 |

### 关键词位置

```
【推荐结构】
emoji + 关键词 + 描述

【示例】
✨油皮亲妈！这款面霜太绝了
   ↑关键词前置
```

## 标题禁忌

| 禁忌 | 原因 | 正确做法 |
|------|------|----------|
| 超过20字 | 显示不全 | 精简到20字内 |
| 无emoji | 不够醒目 | 开头加1-2个emoji |
| 标题党 | 影响信任 | 内容与标题相符 |
| 纯大写 | 视觉疲劳 | 正常大小写 |
| 过多感叹号 | 显得廉价 | 最多1个 |

## 不同品类标题模板

### 美妆护肤

- ✨{肤质}亲妈！{产品}真的绝
- 💄{数字}款{品类}，{效果}
- 🌸{人群}必入！{产品}测评

### 穿搭时尚

- 👗{身材}显瘦穿搭｜{风格}
- 🛍️{场景}穿这套！{效果}
- ✨{季节}必备｜{数字}套{风格}

### 美食探店

- 🍜{城市}必吃！{店名}
- 🔥人均{价格}！{特点}
- ☕{类型}探店｜{推荐}

### 家居生活

- 🏠{户型}改造｜{效果}
- ✨{数字}个收纳技巧，{效果}
- 💡租房必备｜{推荐}
