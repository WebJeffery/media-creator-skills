# 小红书标题模板

## 爆款标题公式

### 1. emoji + 数字 + 品类

```
✨{数字}款{品类}，{效果}！
💡{数字}个{技巧}，{结果}
🔥{数字}步{方法}，{效果}
```

**示例**：
- ✨5款平价面霜，用完皮肤嫩得掐出水！
- 💡3个技巧，让你的妆容持久一整天
- 🔥4步瘦腿法，一周见效

### 2. 人群 + 痛点

```
{emoji}{人群}必看！{解决方案}
{emoji}{人群}的{场景}神器
{emoji}终于找到适合{人群}的{产品}
```

**示例**：
- 💄学生党必看！百元内好用彩妆
- 🏠独居女生的收纳神器
- ✨终于找到适合油皮的面霜

### 3. 对比反转

```
{emoji}从{负面}到{正面}，我只做了{方法}
{emoji}{之前}vs{之后}，差距太大了
{emoji}用了{时间}，{变化}
```

**示例**：
- 🔥从黄黑皮到冷白皮，我只做了这3步
- ✨素颜vs化妆，差距太大了
- 💪坚持30天，腰围小了5cm

### 4. 场景 + 效果

```
{emoji}{场景}绝绝子！{亮点}
{emoji}{场景}穿搭｜{风格}
{emoji}{场景}必备｜{推荐}
```

**示例**：
- 📸约会穿搭绝绝子！男朋友夸了一路
- 🏢通勤穿搭｜简约高级感
- ✈️旅行必备｜行李箱收纳神器

### 5. 紧迫 + 价值

```
{emoji}后悔没早知道的{数字}个{技巧}
{emoji}早该知道的{主题}真相
{emoji}别再{错误}了！{正确做法}
```

**示例**：
- 😭后悔没早知道的10个收纳技巧
- 💡早该知道的护肤真相
- ⚠️别再这样洗脸了！正确方法在这

## 标题优化检查

- [ ] 字数是否在20字以内
- [ ] 是否有吸引眼球的emoji
- [ ] 是否包含核心关键词
- [ ] 是否有数字增加具体感
- [ ] 是否能引发点击欲望
