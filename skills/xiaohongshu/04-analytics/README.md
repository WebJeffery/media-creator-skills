# 小红书 - 分析阶段 Skills

分析阶段帮助创作者了解笔记表现，持续优化创作策略。

## 本阶段目标

- 📊 追踪笔记数据，了解内容表现
- 📈 分析用户行为，优化内容策略
- 🔎 监控竞品动态，学习爆款规律
- 🔮 预测内容趋势，提前布局选题

## 可用 Skills

| Skill | 功能 | 状态 |
|-------|------|------|
| performance-tracker | 笔记数据追踪，监控互动指标 | 🔲 规划中 |
| engagement-analyzer | 互动数据分析，了解用户偏好 | 🔲 规划中 |
| competitor-monitor | 竞品账号监控，学习爆款规律 | 🔲 规划中 |

## 小红书核心数据指标

### 笔记表现指标

| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 曝光量 | 笔记被展示的次数 | - |
| 点击率 | 点击数/曝光量 | 5%+ |
| 互动率 | (点赞+收藏+评论)/曝光量 | 3%+ |
| 点赞数 | 用户点赞次数 | 内容认可度 |
| 收藏数 | 用户收藏次数 | 实用价值度 |
| 评论数 | 用户评论次数 | 话题讨论度 |
| 分享数 | 用户分享次数 | 传播价值度 |

### 数据分析维度

```
┌─────────────────────────────────────────────────┐
│                  笔记分析框架                    │
├─────────────────────────────────────────────────┤
│  封面维度：封面风格 vs 点击率                    │
│  标题维度：标题类型 vs 点击率                    │
│  内容维度：笔记类型 vs 互动率                    │
│  标签维度：标签组合 vs 曝光量                    │
│  时间维度：发布时间 vs 数据表现                  │
└─────────────────────────────────────────────────┘
```

### 小红书流量密码

**高收藏笔记特征**：
- 实用干货、可操作性强
- 清单类、合集类内容
- 教程攻略、步骤清晰

**高点赞笔记特征**：
- 颜值高、视觉吸引
- 情感共鸣、引发认同
- 有趣好玩、轻松愉快

**高评论笔记特征**：
- 话题性强、引发讨论
- 提问互动、邀请参与
- 争议观点、不同看法
